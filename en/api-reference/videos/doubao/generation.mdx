## 请求参数

<ParamField body="model" type="string" required>
  Name of the video generation model

  Supported models:

  * `doubao-seedance-1-0-pro-fast` - Fast version, quick generation (40-90 seconds), suitable for preview and iteration
  * `doubao-seedance-1-0-pro-quality` - High quality version, longer generation time (90-300 seconds), better image quality
</ParamField>

<ParamField body="prompt" type="string" required>
  Video content description

  It is recommended to describe the scene, actions, style, etc. in detail, to get better generation results

  Example: `"Sunset at the seaside, golden sunlight on the sea, waves gently hitting the beach"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  Video duration (seconds)

  Supported range: `2` \~ `12` seconds

  Default: `5`
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  Video aspect ratio

  Optional values:

  * `16:9` - Landscape
  * `9:16` - Portrait
  * `1:1` - Square
  * `4:3` - Traditional ratio
  * `3:4` - Vertical traditional ratio
  * `21:9` - Ultra widescreen

  Default: `16:9`
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  Video resolution

  Optional values:

  * `480p` - Standard definition
  * `720p` - High definition
  * `1080p` - Full HD

  Default: `720p`

  <Warning>
    **1080p limitation**: 1080p resolution is not supported when using reference images (in `image_with_roles` with `role: reference`) scenes
  </Warning>
</ParamField>

## 分辨率与宽高比组合

| Resolution | Supported Aspect Ratios                  | Notes           |
| ---------- | -------------------------------------- | --------------- |
| 480p       | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9       | Fully supported |
| 720p       | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9       | Fully supported |
| 1080p      | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9       | Reference image scenes not supported |

<ParamField body="image_urls" type="array">
  Array of first frame image URLs, used to generate videos based on images

  Used to specify the starting frame of the video

  Example: `["https://example.com/cat.png"]`

  <Warning>
    `image_urls` and `image_with_roles` cannot be used simultaneously
  </Warning>
</ParamField>

<ParamField body="image_with_roles" type="array">
  Array of images with roles, supporting more precise control

  <Expandable title="Field Descriptions">
    <ParamField body="url" type="string" required>
      Image URL address
    </ParamField>

    <ParamField body="role" type="string" required>
      Image role

      Options:

      * `first_frame` - First frame image, serves as the starting frame of the video
      * `last_frame` - Last frame image, serves as the ending frame of the video
      * `reference` - Reference image, used to guide the video style
    </ParamField>
  </Expandable>

  Example:

  ```json 
  [
    {"url": "https://example.com/start.png", "role": "first_frame"},
    {"url": "https://example.com/end.png", "role": "last_frame"}
  ]
  ```

  <Warning>
    `image_urls` and `image_with_roles` cannot be used simultaneously
  </Warning>
</ParamField>

## 使用场景

### Scenario 1: Quickly Generate Landscape Preview Video

```json
{
  "model": "doubao-seedance-1-0-pro-fast",
  "prompt": "Seaside sunset, golden sunlight spilling onto the sea surface, waves gently hitting the beach"
}
```

### Scenario 2: Generate High-Quality Portrait Short Video

```json
{
  "model": "doubao-seedance-1-0-pro-quality",
  "prompt": "A girl spinning under a cherry blossom tree, petals falling with the wind",
  "duration": 5,
  "aspect_ratio": "9:16",
  "resolution": "1080p"
}
```

### Scenario 3: Generate Dynamic Display Video Based on Product Image

```json
{
  "model": "doubao-seedance-1-0-pro-fast",
  "prompt": "Product slowly rotating for display, background pure white, soft lighting",
  "image_urls": ["https://example.com/product.png"],
  "duration": 5,
  "aspect_ratio": "1:1"
}
```

### Scenario 4: Create Dynamic Transition Effects (First and Last Frame)

```json
{
  "model": "doubao-seedance-1-0-pro-quality",
  "prompt": "The scene gradually transitions from day to night, city lights gradually turning on",
  "image_with_roles": [
    {"url": "https://example.com/day.png", "role": "first_frame"},
    {"url": "https://example.com/night.png", "role": "last_frame"}
  ],
  "duration": 5
}
```

<Note>
  **Query Task Result**

  Video generation is an asynchronous task; after submission you will receive a `task_id`. Use the [Get Task Status](/cn/api-reference/tasks/status) endpoint to check progress and results.
</Note>
