   - Asynchronous processing mode, returns task ID for subsequent queries
   - Supports text-to-video and image-to-video (first frame / last frame / multiple reference images)
   - Supports multiple aspect ratios including landscape and portrait

<Note>
This endpoint supports two model variants with identical parameters:
- `seedance2_fast` — Standard fast version
- `seedance2_fast_face` — Supports uploading real human version, features identical to the fast version
</Note>

## Body

<ParamField body="model" type="string" required>
  Model name

  Supports the following model variants (parameters are identical, choose as needed):
  * `seedance2_fast` — Standard fast version
  * `seedance2_fast_face` — Supports uploading real-person version, functions same as fast version
</ParamField>

<ParamField body="params" type="object" required>
  Model parameters object

  <Expandable title="params object properties">
    <ParamField body="prompt" type="string" required>
      Text description for video generation, cannot be empty
    </ParamField>

    <ParamField body="type" type="string" required>
      Reference mode, determines how image/video parameters are used

      * `all_reference` — Smart reference mode: `image_url` used as character/style reference images (up to 9), `video_url` used as reference videos (up to 3)
      * `first_last_frame` — First and last frame mode: `image_url` is the first frame (only 1), `end_image_url` is the last frame, `video_url` not supported

      Required, optional values: `all_reference`, `first_last_frame`
    </ParamField>

    <ParamField body="image_url" type="string[]">
      Reference image URLs

      * `all_reference` mode: up to 9 images, max 50MB each, minimum side length 300px
      * `first_last_frame` mode: only 1 image (first frame), max 10MB, minimum side length 300px
    </ParamField>

    <ParamField body="video_url" type="string[]">
      Reference video URLs

      Supported only in `all_reference` mode. Maximum 3 videos; supports `mp4`, `mov`; single video length 2-15 seconds, total length of all reference videos not exceeding 15 seconds; single video max 50MB; frame rate 24-60 FPS; resolutions supported: `480p`, `720p`; aspect ratio 0.4~2.5; width and height range 300-6000px; total pixels between 409600 and 927408

      <Warning>Only effective in `all_reference` mode, do not pass this parameter when `type` is `first_last_frame`</Warning>
    </ParamField>

    <ParamField body="end_image_url" type="string">
      Last frame image URL

      Max 10MB

      <Note>Only effective in `first_last_frame` mode</Note>
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      Video aspect ratio

      Supported aspect ratios:
      * `16:9` - Landscape (default)
      * `9:16` - Portrait
      * `1:1` - Square
      * `4:3` - Standard
      * `3:4` - Portrait standard
      * `21:9` - Ultra-wide
    </ParamField>

    <ParamField body="resolution" type="string" default="720p">
      Video resolution

      Supported resolutions:
      * `480p`
      * `720p` (default)
    </ParamField>

    <ParamField body="video_length" type="string" default="5">
      Video length (seconds)

      Supported range: 4 to 15 seconds, default 5 seconds
    </ParamField>

    <ParamField body="generate_audio" type="boolean" default="true">
      Whether to generate audio, default `true`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  Client task ID

  User-defined task identifier, required
</ParamField>
