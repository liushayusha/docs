---
title: 'Sora2 Pro Video Generation'
description: |
    - Asynchronous processing mode, returns task ID for subsequent queries
    - Supports text-to-video, image-to-video and other generation modes
    - Professional version, supports higher resolution
api: 'POST https://pxz-api.xinsulv.com/api/openapi/submit'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  All endpoints require authentication using a Bearer Token

  Get the API Key:

  Visit the [API Key management page](https://apimart.ai/console/token) to get your API Key

  Add in the request headers:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" required>
  Model name

  Fixed value: `"sora2_pro"`
</ParamField>

<ParamField body="params" type="object" required>
  Model parameters object

  <Expandable title="params object properties">
    <ParamField body="prompt" type="string" required>
      Text description for video generation

      No limitations
    </ParamField>

    <ParamField body="image_url" type="string">
      URL of the reference image for image-to-video (first frame)

      * Supports only a single image
      * Supports publicly accessible URLs
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      Video aspect ratio

      Supported aspect ratios:

      * `16:9` - Landscape (default)
      * `9:16` - Portrait
    </ParamField>

    <ParamField body="video_length" type="string" default="10">
      Video length (seconds)

      Supported lengths:

      * `10` - 10 seconds (default)
      * `15` - 15 seconds
    </ParamField>

    <ParamField body="resolution" type="string" default="standard">
      Video resolution

      Supported resolutions:

      * `standard` - Standard resolution (default)
      * `high` - High resolution
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="callback_url" type="string">
  Callback URL for task completion notification

  After the task is completed, the system will send a POST request to this URL to notify the result
</ParamField>

<ParamField body="metadata" type="object">
  Custom metadata

  Returned as-is when querying results, can be used for business identification
</ParamField>

## Response

<ResponseField name="task_id" type="string">
  Task ID, used to query the result
</ResponseField>

<ResponseField name="status" type="string">
  Initial status, fixed as `"pending"`
</ResponseField>

<ResponseField name="estimated_credits" type="number">
  Estimated credit consumption
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation time (ISO format)
</ResponseField>
