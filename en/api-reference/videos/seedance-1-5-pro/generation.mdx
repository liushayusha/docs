   - Asynchronous processing mode, returns task ID for subsequent querying
   - Supports text-to-video and image-to-video (first frame / last frame / multiple reference images)
   - Supports multiple aspect ratios including horizontal and vertical

## Body

<ParamField body="model" type="string" required>
  Model Name

  Fixed value: `"seedance_1_5_pro"`
</ParamField>

<ParamField body="params" type="object" required>
  Model Parameters Object

  <Expandable title="params Object Properties">
    <ParamField body="prompt" type="string" required>
      Text Description for Video Generation

      Unlimited
    </ParamField>

    <ParamField body="video_length" type="number" default="5">
      Video Duration (seconds)

      Supported duration range: 4-12 seconds

      Default: 5 seconds
    </ParamField>

    <ParamField body="aspect_ratio" type="string" required>
      Video Aspect Ratio

      Supported aspect ratios:
      * `16:9` - Landscape
      * `9:16` - Portrait
    </ParamField>

    <ParamField body="resolution" type="string" required>
      Video Resolution

      Supported resolutions:
      * `720p` - Standard Definition
      * `1080p` - High Definition
    </ParamField>

    <ParamField body="image_url" type="string">
      Reference Image URL for Image-to-Video (First Frame)

      * Only supports a single image
      * Supports publicly accessible URLs
    </ParamField>

    <ParamField body="end_image_url" type="string">
      Reference Image URL for Ending Frame

      * Only supports a single image
      * Supports publicly accessible URLs
    </ParamField>

    <ParamField body="generate_audio" type="boolean" default="false">
      Whether to Generate Audio

      When set to `true`, the video will include AI-generated accompanying audio

      Default: `false`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  Initiator Task ID

  User-defined task identifier, required
</ParamField>

### Resolution and Aspect Ratio Combinations Table

| Aspect Ratio | Resolution | Actual Size |
|--------|--------|----------|
| 16:9   | 720p   | 1280x720 |
| 16:9   | 1080p  | 1920x1080|
| 9:16   | 720p   | 720x1280 |
| 9:16   | 1080p  | 1080x1920|
