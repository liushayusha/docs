---
title: 'Sora2 Create Character (Character Management)'
description: |
    - Extract characters from videos for subsequent video generation
    - Asynchronous processing mode, returns task ID for later query
    - Videos must contain sound and characters
api: 'POST https://api.apimart.ai/v1/videos/generations'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  All endpoints require authentication using a Bearer Token

  Get the API Key:

  Visit the [API Key management page](https://apimart.ai/console/token) to get your API Key

  Add the following in the request headers:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="sora-2" required>
  Video generation model name

  Supported models:

  * `sora-2` - Standard version
  * `sora-2-pro` - Professional version

  Example: `"sora-2"` or `"sora-2-pro"`
</ParamField>

<ParamField body="timestamps" type="string" required>
  Timestamp range for the character appearance

  Unit is seconds, format is `"start_second,end_second"`

  **‚ö†Ô∏è Limitations:**

  * Minimum time range difference is 1 second
  * Maximum time range difference is 3 seconds

  Example: `"1,3"` means the character appears in the video from second 1 to second 3
</ParamField>

<ParamField body="url" type="string">
  URL of the video containing the character to be created

  **‚ö†Ô∏è Requirements:**

  * Video must contain audio
  * Video must contain characters

  **üí° Note:**

  * Either `url` or `from_task` must be set, one of them is required

  Example: `"https://upload.apimart.ai/f/video/9998234807907072-ad4c2656-94a4-44d4-9eaa-5be2855a7798-video_task_01KBYT63R1TGZNT4EHAD495ZJ3.mp4"`
</ParamField>

<ParamField body="from_task" type="string">
  ID of the already generated video task

  You can create a character based on an already generated task ID

  **üí° Note:**

  * Either `url` or `from_task` must be set, one of them is required
  * Using this parameter allows extracting characters directly from a previously generated video task

  Example: `"task_01KBYT59JDHB4A3KDDR9N9JVWP"`
</ParamField>

## Response

<ResponseField name="code" type="integer">
  Response status code, 200 on success
</ResponseField>

<ResponseField name="data" type="array">
  Returned data array

  <Expandable title="Array Elements">
    <ResponseField name="status" type="string">
      Task status, initially `submitted` upon submission
    </ResponseField>

    <ResponseField name="task_id" type="string">
      Unique identifier for the task, used to query task status and results

      Once created, this character task ID can be referenced via the `character_url` parameter when generating videos
    </ResponseField>
  </Expandable>
</ResponseField>
