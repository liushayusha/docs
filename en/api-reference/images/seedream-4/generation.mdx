## Authorizations

<ParamField header="Authorization" type="string" required>

  Get API Key:

  Visit [API Key Management Page](https://apimart.ai/console/token) to get your API Key.

  Add the following to request headers:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="doubao-seedance-4-0" required>
  Image generation model name

  Example: `"doubao-seedance-4-0"`
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description for image generation

  Maximum 1000 characters
</ParamField>

<ParamField body="size" type="string" default="1:1">
  Aspect ratio of the generated image

  Supported aspect ratios:

  * `1:1` - Square (default)
  * `4:3` - Landscape 4:3
  * `3:4` - Portrait 3:4
  * `16:9` - Landscape widescreen
  * `9:16` - Portrait vertical screen
  * `3:2` - Landscape 3:2
  * `2:3` - Portrait 2:3
  * `21:9` - Ultra widescreen
  * `9:21` - Ultra vertical screen
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  Image resolution

  Supported resolutions:

  * `1K` - Basic resolution
  * `2K` - Standard resolution (default)
  * `4K` - High definition resolution

  **Resolution reference sizes:**

  | Resolution | 1:1 Size    | 16:9 Size   |
  | --- | --------- | --------- |
  | 1K  | 1024x1024 | 1280x720  |
  | 2K  | 2048x2048 | 2560x1440 |
  | 4K  | 4096x4096 | 5404x3040 |

  <Expandable title="Full Pixel Mapping Table">

</ParamField>

<ParamField body="n" type="integer">
  Number of images to generate

  Value range: 1-15 (minimum 1, maximum 15)

  Default: 1

  **Note:**

  * The actual number of images generated is affected both by this parameter and by the number of input reference images. The sum of input reference images + final generated images ≤ 15
  * **Must input a pure number (e.g., `1`), do not add quotes, or it will cause an error**

  The charge will be deducted based on the quantity generated
</ParamField>

<ParamField body="image_urls" type="array">
  List of reference image URLs for image-to-image generation or image editing

  Supports two formats:

  **1. Full image URL**

  * Publicly accessible image URLs (http:// or https://)
  * Example: `https://example.com/image.jpg`

  **2. Base64 encoding format**

  * **Must use full Data URI format**
  * Format: `data:image/{format};base64,{base64data}`
  * Supported image formats: jpeg, png
  * Example: `data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
  * ⚠️ Note: Must include the prefix `data:image/jpeg;base64,`

  **Limits:**

  * Maximum 10 images
  * Image formats: jpeg, png
  * Aspect ratio (width/height) range: [1/3, 3]
  * Width or height (px) > 14
  * Single image size: no more than 10MB
  * Total pixels: no more than 6000×6000 px
</ParamField>

<ParamField body="optimize_prompt_options" type="string" default="standard">
  Prompt optimization mode

  * `standard`: Standard mode, higher quality of generated content, longer processing time
  * `fast`: Fast mode, shorter processing time, average quality

  Default: `standard`
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  Sequential image generation mode (exclusive feature of Doubao)

  Controls whether to generate multiple images:

  * `disabled`: Disable sequential mode, generate only 1 image even if there are multiple reference images (default)
  * `auto`: Enable sequential mode to generate multiple images

  **Usage instructions:**

  * ✅ Must provide `image_urls` - at least 1 reference image required
  * ✅ Set `n: 3` or use `sequential_image_generation: "auto"` + `max_images: 3`
  * ✅ This will generate 3 different images based on the reference images
  * ⚠️ When `n > 1`, it will automatically set to `auto`

  **Limitations:**

  * Pure text-to-image (no `image_urls`) cannot generate multiple images, this is a limitation of the Doubao API
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  Sequential image generation options

  Available when `sequential_image_generation` is set to `auto`

  **Properties:**

  * `max_images` (integer): Specifies the number of images to generate, range: 1-15

  **Example:**

  ```json 
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 3
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  Whether to add a watermark on the generated images

  * `true`: Add watermark
  * `false`: Do not add watermark (default)
</ParamField>
