---
title: 'Get Task Status'
description: |
    - Query the execution status and result of asynchronous tasks
    - Real-time status updates and progress tracking
    - Obtain generated results upon task completion
    - Support multilingual returns (zh/en/ko/ja)
api: 'GET https://api.apimart.ai/v1/tasks/{task_id}'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  All endpoints require authentication using a Bearer Token

  Get the API Key:

  Visit the [API Key management page](https://apimart.ai/console/token) to get your API Key

  Add to the request header when using:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Path parameters

<ParamField path="task_id" type="string" required>
  Task ID returned by the generate API
</ParamField>

## Query parameters

<ParamField query="language" type="string">
  The language of the returned content. Supported values:

  * `zh` - Chinese
  * `en` - English
  * `ko` - Korean
  * `ja` - Japanese

  Defaults to English
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier of the task
</ResponseField>

<ResponseField name="status" type="string">
  Task status values:

  * `pending` - queued and waiting to be processed
  * `processing` - processing
  * `completed` - successfully completed
  * `failed` - failed
  * `cancelled` - cancelled by user
</ResponseField>

<ResponseField name="progress" type="integer">
  Task progress percentage (0â€“100)
</ResponseField>

<ResponseField name="result" type="object">
  Task result, returned only when status is `completed`

  <Expandable title="Properties">
    <ResponseField name="images" type="array">
      Array of generated image objects (image generation tasks)
    </ResponseField>

    <ResponseField name="videos" type="array">
      Array of generated video objects (video generation tasks)
    </ResponseField>

    <ResponseField name="thumbnail_url" type="string">
      Video thumbnail URL (video generation tasks)

      <Note>
        **Important:** This field is returned only if thumbnail generation was explicitly requested in the task creation parameters. If thumbnail generation was not requested, this field will not be included in the response.
      </Note>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created" type="integer">
  Task creation timestamp
</ResponseField>

<ResponseField name="completed" type="integer">
  Task completion timestamp (exists only when completed)
</ResponseField>

<ResponseField name="estimated_time" type="integer">
  Estimated completion time (seconds)
</ResponseField>

<ResponseField name="actual_time" type="integer">
  Actual completion time (seconds) (exists only when completed)
</ResponseField>

<ResponseField name="error" type="object">
  Error details (exists only when status is `failed`)

  <Expandable title="Properties">
    <ResponseField name="code" type="integer">
      Error code
    </ResponseField>

    <ResponseField name="message" type="string">
      Error message
    </ResponseField>

    <ResponseField name="type" type="string">
      Error type
    </ResponseField>
  </Expandable>
</ResponseField>
