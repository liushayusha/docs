## Authorizations

<ParamField header="Authorization" type="string" required>
  모든 API는 Bearer Token 인증이 필요합니다.

  API 키 받기:

  [API Key 관리 페이지](https://apimart.ai/console/token)에 접속하여 API 키를 받으세요.

  요청 헤더에 다음을 추가하여 사용합니다:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Path parameters

<ParamField path="task_id" type="string" required>
  API가 반환하는 작업 ID 생성
</ParamField>

## Query parameters

<ParamField query="language" type="string">
  반환 내용의 언어로, 다음 값을 지원합니다:

  * `zh` - 중국어
  * `en` - 영어
  * `ko` - 한국어
  * `ja` - 일본어

  기본값은 영어입니다
</ParamField>

## Response

<ResponseField name="id" type="string">
  작업 고유 식별자
</ResponseField>

<ResponseField name="status" type="string">
  작업 상태 값:

  * `pending` - 대기 중
  * `processing` - 처리 중
  * `completed` - 성공적으로 완료
  * `failed` - 실패
  * `cancelled` - 사용자에 의해 취소됨
</ResponseField>

<ResponseField name="progress" type="integer">
  작업 진행률 백분율 (0–100)
</ResponseField>

<ResponseField name="result" type="object">
  작업 결과, 상태가 `completed`일 때만 반환

  <Expandable title="Properties">
    <ResponseField name="images" type="array">
      생성된 이미지 객체 배열 (이미지 생성 작업)
    </ResponseField>

    <ResponseField name="videos" type="array">
      생성된 동영상 객체 배열 (동영상 생성 작업)
    </ResponseField>

    <ResponseField name="thumbnail_url" type="string">
      동영상 썸네일 URL (동영상 생성 작업)

      <Note>
        **중요:** 이 필드는 생성 작업 요청 시 썸네일 생성이 명시적으로 지정된 경우에만 반환됩니다. 요청에 썸네일 생성이 지정되지 않은 경우, 응답에 이 필드가 포함되지 않습니다.
      </Note>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created" type="integer">
  작업 생성 타임스탬프
</ResponseField>

<ResponseField name="completed" type="integer">
  작업 완료 타임스탬프 (완료 시에만 존재)
</ResponseField>

<ResponseField name="estimated_time" type="integer">
  예상 완료 시간 (초)
</ResponseField>

<ResponseField name="actual_time" type="integer">
  실제 완료 시간 (초) (완료 시에만 존재)
</ResponseField>

<ResponseField name="error" type="object">
  오류 세부 정보 (상태가 `failed`일 때만 존재)

  <Expandable title="Properties">
    <ResponseField name="code" type="integer">
      오류 코드
    </ResponseField>

    <ResponseField name="message" type="string">
      오류 메시지
    </ResponseField>

    <ResponseField name="type" type="string">
      오류 유형
    </ResponseField>
  </Expandable>
</ResponseField>
