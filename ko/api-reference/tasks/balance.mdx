## Authorizations

<ParamField header="Authorization" type="string" required>
  모든 API는 Bearer Token 인증이 필요합니다.

  API Key 받기:

  <a href={apiKeyUrl} target="_blank">API Key 관리 페이지</a>에 방문하여 API Key를 받으세요.

  요청 헤더에 다음을 추가하여 사용하세요:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Response

<ResponseField name="statusCode" type="number">
  HTTP 상태 코드, 200은 성공을 의미합니다
</ResponseField>

<ResponseField name="message" type="string">
  메시지, 성공 시 빈 문자열
</ResponseField>

<ResponseField name="data" type="object">
  반환 데이터

  <Expandable title="Properties">
    <ResponseField name="credits" type="number">
      사용자의 현재 크레딧 잔액
    </ResponseField>

    <ResponseField name="free_quotas" type="array">
      각 모델의 무료 할당량 정보

      <Expandable title="Array Element Properties">
        <ResponseField name="model_key" type="string">
          모델 식별자
        </ResponseField>

        <ResponseField name="total" type="number">
          무료 할당의 총 횟수
        </ResponseField>

        <ResponseField name="used" type="number">
          사용된 횟수
        </ResponseField>

        <ResponseField name="remaining" type="number">
          남은 사용 가능 횟수
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="ok" type="boolean">
  요청 성공 여부
</ResponseField>

  ```go Go
  package main

  import (
      "fmt"
      "io/ioutil"
      "net/http"
  )

  func main() {
      url := "https://aireiter.com/api/openapi/balance"

      req, _ := http.NewRequest("GET", url, nil)
      req.Header.Set("Authorization", "Bearer <token>")

      client := &http.Client{}
      resp, err := client.Do(req)
      if err != nil {
          panic(err)
      }
      defer resp.Body.Close()

      body, _ := ioutil.ReadAll(resp.Body)
      fmt.Println(string(body))
  }
  ```

  ```java Java
  import java.io.BufferedReader;
  import java.io.InputStreamReader;
  import java.net.HttpURLConnection;
  import java.net.URL;

  public class Main {
      public static void main(String[] args) {
          try {
              URL url = new URL("https://aireiter.com/api/openapi/balance");
              HttpURLConnection conn = (HttpURLConnection) url.openConnection();
              conn.setRequestMethod("GET");
              conn.setRequestProperty("Authorization", "Bearer <token>");

              BufferedReader br = new BufferedReader(new InputStreamReader(conn.getInputStream()));
              String line;
              StringBuilder response = new StringBuilder();
              while ((line = br.readLine()) != null) {
                  response.append(line);
              }
              br.close();

              System.out.println(response.toString());
          } catch (Exception e) {
              e.printStackTrace();
          }
      }
  }
  ```

  ```swift Swift
  import Foundation

  let url = URL(string: "https://aireiter.com/api/openapi/balance")!
  var request = URLRequest(url: url)
  request.httpMethod = "GET"
  request.setValue("Bearer <token>", forHTTPHeaderField: "Authorization")

  let task = URLSession.shared.dataTask(with: request) { data, response, error in
      if let error = error {
          print("Error: \(error)")
          return
      }

      if let data = data, let responseString = String(data: data, encoding: .utf8) {
          print(responseString)
      }
  }

  task.resume()
  ```
