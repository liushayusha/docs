## Body

<ParamField body="model" type="string" required>
  모델 이름

  고정 값: `"seedance_pro"`
</ParamField>

<ParamField body="params" type="object" required>
  모델 파라미터 객체

  <Expandable title="params 객체 속성">
    <ParamField body="prompt" type="string" required>
      영상 생성을 위한 텍스트 설명

      제한 없음
    </ParamField>

    <ParamField body="video_length" type="number" default="5">
      영상 길이(초)

      지원 길이 범위: 4-12초

      기본값: 5초
    </ParamField>

    <ParamField body="aspect_ratio" type="string" required>
      영상 가로세로 비율

      지원 비율:
      * `16:9` - 가로
      * `9:16` - 세로
    </ParamField>

    <ParamField body="resolution" type="string" required>
      영상 해상도

      지원 해상도:
      * `720p` - 표준 화질
      * `1080p` - 고화질
    </ParamField>

    <ParamField body="image_url" type="string">
      영상 생성용 참조 이미지 URL(첫 프레임)

      * 단일 이미지만 지원
      * 공개 접근 가능한 URL 지원
    </ParamField>

    <ParamField body="end_image_url" type="string">
      마지막 프레임 참조 이미지 URL

      * 단일 이미지만 지원
      * 공개 접근 가능한 URL 지원
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  요청자 작업 ID

  사용자 정의 작업 식별자, 필수 입력
</ParamField>

### 해상도 및 가로세로 비율 조합 표

| 가로세로 비율 | 해상도 | 실제 크기 |
|--------|--------|----------|
| 16:9   | 720p   | 1280x720 |
| 16:9   | 1080p  | 1920x1080|
| 9:16   | 720p   | 720x1280 |
| 9:16   | 1080p  | 1080x1920|

              os.flush();
              os.close();

              BufferedReader br = new BufferedReader(new InputStreamReader(conn.getInputStream()));
              String line;
              StringBuilder response = new StringBuilder();
              while ((line = br.readLine()) != null) {
                  response.append(line);
              }
