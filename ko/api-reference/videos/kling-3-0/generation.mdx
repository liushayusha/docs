## Body

<ParamField body="model" type="string" required>
  모델 이름:

  * `"kling_3_0"` - 기본값
</ParamField>

<ParamField body="params" type="object" required>
  모델 매개변수 객체

  <Expandable title="params 객체 속성">
    <ParamField body="prompt" type="string" required>
      비디오 생성 텍스트 설명

      * 필수
    </ParamField>

    <ParamField body="image_url" type="string">
      참조 이미지 URL, 영상 생성 첫 프레임 용

      * 단일 이미지만 지원
      * 공개적으로 액세스 가능한 URL만 지원
    </ParamField>

    <ParamField body="end_image_url" type="string">
      마지막 프레임 참조 이미지 URL

      * 단일 이미지만 지원
      * 첫 프레임 image_url과 함께 사용해야 함
      * 공개적으로 액세스 가능한 URL만 지원
    </ParamField>

    <ParamField body="resolution" type="string" default="1080p">
      비디오 해상도

      * `720p` - 표준 화질
      * `1080p` - 고화질 (기본값)
      * `4K` - 초고화질
    </ParamField>

    <ParamField body="video_length" type="integer" default="5">
      비디오 길이(초)

      * 허용 범위: 3 ~ 15 (정수)
    </ParamField>

    <ParamField body="generate_audio" type="boolean" default="true">
      화면과 동기화된 AI 오디오 생성 여부

      * `true` - 오디오 생성 (기본값)
      * `false` - 오디오 생성 안 함
      * 4K 해상도에서 오디오 별도 과금 없음
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      비디오 가로세로 비율

      * `16:9` - 가로 (기본값)
      * `9:16` - 세로
      * `1:1` - 정사각형
      * 참조 이미지 제공 시 이 매개변수는 생략 가능하며 시스템이 자동으로 비율을 조정함
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  발신자 작업 ID

  사용자가 정의한 작업 식별자, 필수
</ParamField>
