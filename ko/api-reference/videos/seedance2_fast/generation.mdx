## Body

<ParamField body="model" type="string" required>
  모델 이름

  고정값: `"seedance2_fast"`
</ParamField>

<ParamField body="params" type="object" required>
  모델 파라미터 객체

  <Expandable title="params 객체 속성">
    <ParamField body="prompt" type="string" required>
      비디오 생성을 위한 텍스트 설명으로, 비어 있을 수 없습니다
    </ParamField>

    <ParamField body="type" type="string" required>
      참조 모드로, 이미지/비디오 파라미터 사용 방식을 결정합니다

      * `all_reference` — 스마트 참조 모드: `image_url`은 캐릭터/스타일 참조 이미지(최대 9장), `video_url`은 참조 비디오(최대 3개)
      * `first_last_frame` — 시작과 끝 프레임 모드: `image_url`은 시작 프레임(1장만), `end_image_url`은 끝 프레임이며, `video_url`은 지원하지 않음

      필수이며, 선택 가능 값: `all_reference`, `first_last_frame`
    </ParamField>

    <ParamField body="image_url" type="string[]">
      참조 이미지 URL

      * `all_reference` 모드: 최대 9장, 각 파일 최대 50MB, 최소 가로세로 300px
      * `first_last_frame` 모드: 1장(시작 프레임만), 최대 10MB, 최소 가로세로 300px
    </ParamField>

    <ParamField body="video_url" type="string[]">
      참조 비디오 URL

      `all_reference` 모드만 지원. 최대 3개; 형식 `mp4`, `mov` 지원; 개별 비디오 길이 2-15초, 전체 참조 비디오 총 길이 15초 이하; 개별 비디오 최대 50MB; 프레임 속도 24-60 FPS; 해상도 `480p`, `720p` 지원; 가로세로 비율 0.4~2.5; 가로세로 픽셀 범위 300-6000px; 전체 픽셀 수는 409600-927408 사이여야 함

      <Warning>`all_reference` 모드에서만 유효하며, `type`이 `first_last_frame`일 때는 이 파라미터를 전달하지 마세요</Warning>
    </ParamField>

    <ParamField body="end_image_url" type="string">
      끝 프레임 이미지 URL

      최대 10MB

      <Note>`first_last_frame` 모드에서만 유효</Note>
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      비디오 가로세로 비율

      지원 가로세로 비율:
      * `16:9` - 가로 화면(기본값)
      * `9:16` - 세로 화면
      * `1:1` - 정사각형
      * `4:3` - 표준
      * `3:4` - 세로 표준
      * `21:9` - 울트라와이드
    </ParamField>

    <ParamField body="resolution" type="string" default="720p">
      비디오 해상도

      지원 해상도:
      * `480p`
      * `720p`(기본값)
    </ParamField>

    <ParamField body="video_length" type="string" default="5">
      비디오 길이(초)

      지원 길이 범위: 4~15초, 기본값 5초
    </ParamField>

    <ParamField body="generate_audio" type="boolean" default="true">
      오디오 생성 여부, 기본값 `true`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  요청자 작업 ID

  사용자가 정의하는 작업 식별자, 필수
</ParamField>
