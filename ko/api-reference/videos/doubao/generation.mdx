## 요청 매개변수

<ParamField body="model" type="string" required>
  비디오 생성 모델 이름

  지원되는 모델:

  * `doubao-seedance-1-0-pro-fast` - 빠른 버전, 생성 속도 빠름 (40-90초), 미리보기 및 반복에 적합
  * `doubao-seedance-1-0-pro-quality` - 고품질 버전, 생성 시간이 더 김 (90-300초), 화질 우수
</ParamField>

<ParamField body="prompt" type="string" required>
  비디오 내용 설명

  더 나은 생성 효과를 위해 장면, 동작, 스타일 등을 자세하게 설명하는 것이 좋습니다

  예시: `"바닷가 일몰, 황금빛 햇살이 바다 위에 내려앉고, 파도가 부드럽게 해변을 두드림"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  비디오 길이 (초)

  지원 범위: `2` ~ `12` 초

  기본값: `5`
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  비디오 종횡비

  선택 값:

  * `16:9` - 가로 화면
  * `9:16` - 세로 화면
  * `1:1` - 정사각형
  * `4:3` - 전통 비율
  * `3:4` - 세로 전통 비율
  * `21:9` - 울트라 와이드 화면

  기본값: `16:9`
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  비디오 해상도

  선택 값:

  * `480p` - 표준 화질
  * `720p` - 고화질
  * `1080p` - 풀 HD

  기본값: `720p`

  <Warning>
    **1080p 제한**: 참조 이미지(`image_with_roles` 내 `role: reference`)를 사용하는 장면에서는 1080p 해상도를 지원하지 않습니다
  </Warning>
</ParamField>

## 해상도와 종횡비 조합

| 해상도   | 지원하는 종횡비                            | 비고          |
| ----- | --------------------------------- | -------- |
| 480p  | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9  | 모두 지원      |
| 720p  | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9  | 모두 지원      |
| 1080p | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9  | 참조 이미지 장면은 미지원 |

<ParamField body="image_urls" type="array">
  첫 프레임 이미지 URL 배열로, 이미지를 기반으로 비디오를 생성

  비디오의 시작 화면을 지정하는 데 사용

  예시: `["https://example.com/cat.png"]`

  <Warning>
    `image_urls`와 `image_with_roles`는 동시에 사용할 수 없습니다
  </Warning>
</ParamField>

<ParamField body="image_with_roles" type="array">
  역할이 지정된 이미지 배열로, 보다 세밀한 제어가 가능

  <Expandable title="필드 설명">
    <ParamField body="url" type="string" required>
      이미지 URL 주소
    </ParamField>

    <ParamField body="role" type="string" required>
      이미지 역할

      선택 가능한 값:

      * `first_frame` - 첫 프레임 이미지로, 비디오 시작 화면
      * `last_frame` - 마지막 프레임 이미지로, 비디오 종료 화면
      * `reference` - 참조 이미지로, 비디오 스타일 가이드용
    </ParamField>
  </Expandable>

  예시:

  ```json 
  [
    {"url": "https://example.com/start.png", "role": "first_frame"},
    {"url": "https://example.com/end.png", "role": "last_frame"}
  ]
  ```

  <Warning>
    `image_urls`와 `image_with_roles`는 동시에 사용할 수 없습니다
  </Warning>
</ParamField>

## 사용 사례

### 사례 1: 빠른 가로형 미리보기 비디오 생성

```json
{
  "model": "doubao-seedance-1-0-pro-fast",
  "prompt": "바닷가 일몰, 금빛 햇살이 바다 위에 비치고, 파도가 부드럽게 모래사장을 때립니다"
}
```

### 사례 2: 고품질 세로형 짧은 비디오 생성

```json
{
  "model": "doubao-seedance-1-0-pro-quality",
  "prompt": "한 소녀가 벚꽃 나무 아래에서 빙글빙글 돌고, 꽃잎이 바람에 흩날립니다",
  "duration": 5,
  "aspect_ratio": "9:16",
  "resolution": "1080p"
}
```

### 사례 3: 제품 이미지를 기반으로 한 동적 전시 비디오 생성

```json
{
  "model": "doubao-seedance-1-0-pro-fast",
  "prompt": "제품이 천천히 회전하면서 전시되고, 배경은 순백색이며 빛이 부드럽습니다",
  "image_urls": ["https://example.com/product.png"],
  "duration": 5,
  "aspect_ratio": "1:1"
}
```

### 사례 4: 동적 전환 효과 제작 (처음과 마지막 프레임)

```json
{
  "model": "doubao-seedance-1-0-pro-quality",
  "prompt": "화면이 낮에서 밤으로 점차 전환되고, 도시의 불빛이 서서히 켜집니다",
  "image_with_roles": [
    {"url": "https://example.com/day.png", "role": "first_frame"},
    {"url": "https://example.com/night.png", "role": "last_frame"}
  ],
  "duration": 5
}
```
