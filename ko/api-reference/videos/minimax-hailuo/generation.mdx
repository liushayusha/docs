## 인증

<ParamField header="Authorization" type="string" required>
  모든 엔드포인트는 Bearer Token 인증이 필요합니다.

  API 키 받기:

  [API Key 관리 페이지](https://apimart.ai/console/token)에서 API 키를 확인하세요.

  사용 시 요청 헤더에 다음을 추가합니다:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## 요청 매개변수

<ParamField body="model" type="string" required>
  비디오 생성 모델 이름

  고정값: `MiniMax-Hailuo-02`
</ParamField>

<ParamField body="prompt" type="string" required>
  비디오 내용 설명

  더 나은 생성 효과를 위해 장면, 동작, 스타일 등을 자세히 설명하는 것이 좋습니다

  예시: `"귀여운 고양이가 잔디밭에서 달리고 있다"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  비디오 길이(초)

  선택 가능한 값:

  * `5` - 5초 비디오
  * `10` - 10초 비디오

  기본값: `5`

  <Warning>
    **1080p 제한**: 1080p 해상도 사용 시 5초 길이만 지원됩니다
  </Warning>
</ParamField>

<ParamField body="metadata" type="object">
  비디오 생성 고급 구성 옵션

  <Expandable title="metadata 필드 설명">
    <ParamField body="metadata.resolution" type="string" default="768p">
      비디오 해상도

      선택 가능한 값:

      * `512p` - 표준 화질
      * `768p` - 고화질
      * `1080p` - 풀 HD (6초 길이만 지원)

      기본값: `768p`
    </ParamField>

    <ParamField body="metadata.prompt_optimizer" type="boolean" default="true">
      프롬프트 자동 최적화 여부

      활성화하면 시스템이 더 나은 생성 결과를 위해 프롬프트를 자동으로 최적화합니다

      기본값: `true`
    </ParamField>

    <ParamField body="metadata.fast_pretreatment" type="boolean" default="false">
      프롬프트 최적화 시간 단축 여부

      활성화하면 처리 속도가 빨라지지만 최적화 효과에 다소 영향이 있을 수 있습니다

      기본값: `false`
    </ParamField>

    <ParamField body="metadata.watermark" type="boolean" default="false">
      워터마크 추가 여부

      기본값: `false`
    </ParamField>

    <ParamField body="metadata.first_frame_image" type="string">
      비디오 시작 프레임 이미지

      두 가지 형식 지원:

      * **공용 URL**: `https://example.com/start.jpg`
      * **Base64 인코딩**: `data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      비디오 시작 화면 지정에 사용
    </ParamField>

    <ParamField body="metadata.last_frame_image" type="string">
      비디오 종료 프레임 이미지

      두 가지 형식 지원:

      * **공용 URL**: `https://example.com/end.jpg`
      * **Base64 인코딩**: `data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      비디오 종료 화면 지정에 사용
    </ParamField>
  </Expandable>
</ParamField>

## Parameter Limitations

| Limit        | Description                                           |
| --------- | -------------------------------------------------- |
| Duration     | Only 5 or 10 seconds supported                        |
| 1080p Resolution | Only supported for 5-second duration                  |
| Image Format | Supports public network URL or Base64 encoding (`data:image/jpeg;base64,...`) |

## 해상도와 길이 조합

| 해상도   | 지원 길이     | 비고       |
| ----- | -------- | -------- |
| 512p  | 5초, 10초  | 모두 지원    |
| 768p  | 5초, 10초  | 모두 지원    |
| 1080p | 5초       | 10초 불가   |
