---
title: 'Sora2 비디오 편집'
description: |
    - 생성된 비디오를 기반으로 2차 편집
    - 비동기 처리 모드, 후속 조회를 위한 작업 ID 반환
    - 생성된 비디오 링크 유효 기간은 24시간이므로 빠르게 저장하세요
api: 'POST https://api.apimart.ai/v1/{video_id}/remix'
---
## 경로 매개변수

<ParamField path="video_id" type="string" required>
  원본 비디오 작업 ID

  이전에 비디오 생성 API를 통해 반환된 작업 ID입니다
</ParamField>

## Authorizations

<ParamField header="Authorization" type="string" required>
  모든 API 엔드포인트는 Bearer Token 인증이 필요합니다

  API Key를 얻는 방법:

  [API Key 관리 페이지](https://apimart.ai/console/token)로 이동하여 API Key를 받으세요

  요청 헤더에 추가하세요:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="sora-2" required>
  비디오 편집 모델 이름

  지원되는 모델:

  * `sora-2` - 표준 버전
  * `sora-2-pro` - 전문 버전, 더 긴 길이 지원

  예시: `"sora-2"` 또는 `"sora-2-pro"`
</ParamField>

<ParamField body="prompt" type="string" required>
  편집 명령 설명
</ParamField>

<ParamField body="duration" type="integer">
  비디오 길이(초)

  * `sora-2`：10초 또는 15초 지원
  * `sora-2-pro`：15초(고화질) 또는 25초 지원

  예시: `15`
</ParamField>

<ParamField body="aspect_ratio" type="string">
  비디오 가로세로 비율

  지원 비율:

  * `16:9` - 표준 와이드스크린(추천)
  * `9:16` - 세로 화면 모드
  * `1:1` - 정사각형

  예시: `"16:9"`
</ParamField>

          const char *url = "https://api.apimart.ai/v1/videos/{video_id}/remix";

              "\"prompt\":\"강아지 한 마리 더 추가\","

          struct curl_slist *headers = NULL;

          headers = curl_slist_append(headers, "Content-Type: application/json");

          curl_easy_setopt(curl, CURLOPT_URL, url);
          curl_easy_setopt(curl, CURLOPT_POSTFIELDS, payload);
          curl_easy_setopt(curl, CURLOPT_HTTPHEADER, headers);

          res = curl_easy_perform(curl);

              fprintf(stderr, "curl_easy_perform() failed: %s\n",
                      curl_easy_strerror(res));
          }

          curl_slist_free_all(headers);
          curl_easy_cleanup(curl);
      }

      curl_global_cleanup();
      return 0;
  }
