---
title: 'VEO3.1 빠른 영상 생성'
description: |
    - 비동기 처리 모드, 후속 조회를 위한 작업 ID 반환
    - 텍스트에서 영상 변환, 이미지에서 영상 생성 등 지원
    - 속도 최적화된 빠른 버전
    - 첫 프레임 및 마지막 프레임 참고 이미지 지원
api: 'POST https://pxz-api.xinsulv.com/api/openapi/submit'

## Authorizations

<ParamField header="Authorization" type="string" required>
  모든 엔드포인트는 Bearer Token을 사용한 인증이 필요합니다.

  API Key 받기:

  [API Key 관리 페이지](https://apimart.ai/console/token)에서 API Key를 받아주세요.

  사용 시 요청 헤더에 다음을 추가하세요:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" required>
  모델 이름

  고정값: `"veo3_1_fast"`
</ParamField>

<ParamField body="params" type="object" required>
  모델 파라미터 객체

  <Expandable title="params 객체 속성">
    <ParamField body="prompt" type="string" required>
      영상 생성을 위한 텍스트 설명

      제한 없음
    </ParamField>

    <ParamField body="image_url" type="string">
      영상 생성 시 참고하는 이미지 URL (첫 프레임)

      * 단일 이미지만 지원
      * 공개적으로 접근 가능한 URL을 지원
    </ParamField>

    <ParamField body="end_image_url" type="string">
      마지막 프레임 참고 이미지 URL

      * 단일 이미지만 지원
      * 공개적으로 접근 가능한 URL을 지원
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      영상 가로세로 비율

      지원하는 가로세로 비율:

      * `16:9` - 가로형 (기본값)
      * `9:16` - 세로형
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="callback_url" type="string">
  작업 완료 후 콜백 알림 주소

  작업이 완료되면 시스템이 해당 주소로 POST 요청을 보내 결과를 알립니다
</ParamField>

<ParamField body="metadata" type="object">
  커스텀 메타데이터

  조회 시 원본 그대로 반환되며 비즈니스 식별용으로 사용할 수 있습니다
</ParamField>

</ResponseField>
