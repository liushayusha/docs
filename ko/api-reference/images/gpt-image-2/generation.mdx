      참고 이미지 배열, 전달 시 이미지 생성 모드 적용

      * 최대 9장까지 참고 이미지 허용
      * 다음 두 가지 형식 지원:
        1. 공개 접근 가능한 URL
        2. Base64 인코딩 형식
           * 완전한 Data URI 형식 사용 필수: `data:image/{포맷};base64,{base64 데이터}`
           * 지원하는 이미지 포맷: jpeg, png, webp
           * 예시: `data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
           * ⚠️ 주의: `data:image/jpeg;base64,` 접두어 포함 필수
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="auto">
      이미지 생성 비율

      기본값은 `resolution`에 따라 다름:

      * `resolution`이 `1K` / `2K`일 때 미전달하면 기본값은 `auto`
      * `resolution`이 `4K`일 때 미전달하면 기본값은 `16:9`

      다음 값 지원:

      * `auto` - 가장 적합한 비율 자동 선택
      * `1:1` - 정사각형
      * `16:9` / `9:16` - 와이드 가로 / 세로
      * `4:3` / `3:4` - 표준 가로 / 세로
      * `3:2` / `2:3` - 클래식 가로 / 세로
      * `5:4` / `4:5` - 근사 가로 / 세로
      * `2:1` / `1:2` - 넓은 가로 / 긴 세로
      * `21:9` / `9:21` - 초광폭 가로 / 초긴 세로

      <Warning>
        비율 문자열 또는 `auto`만 지원; 픽셀 치수(예: `1024x1024`) 전달 시 오류 발생
      </Warning>
    </ParamField>

    <ParamField body="resolution" type="string" default="1K">
      해상도 단계

      실제 출력 이미지 선명도 제어:

      * `1K` - 1024 기준, 비용 절감용 일상 사용에 적합 (기본값)
      * `2K` - 2048 기준, 포스터 / 고화질 요구에 적합
      * `4K` - 3840 기준, **6개 비율만 지원** (`16:9` / `9:16` / `2:1` / `1:2` / `21:9` / `9:21`)

      <Warning>
        허용되지 않는 4K 조합은 400 오류 반환:

        * `1:1` × `4K` ❌ (픽셀 초과 제한)
        * `3:2` / `2:3` × `4K` ❌ (픽셀 초과 제한)
        * `4:3` / `3:4` × `4K` ❌ (픽셀 초과 제한)
        * `5:4` / `4:5` × `4K` ❌ (픽셀 초과 제한)

        위 비율에서 고화질을 원하면 `resolution=2K` 사용 권장.
      </Warning>
    </ParamField>

## 크기 × 해상도 매핑 표

`aspect_ratio × resolution` → 실제 픽셀 (13 비율 × 3 단계):

| aspect\_ratio | `1K`      | `2K`      | `4K`          |
| ------------- | --------- | --------- | ------------- |
| `1:1`         | 1024×1024 | 2048×2048 | ❌ 최대 픽셀 초과       |
| `3:2`         | 1536×1024 | 2048×1360 | ❌ 최대 픽셀 초과       |
| `2:3`         | 1024×1536 | 1360×2048 | ❌ 최대 픽셀 초과       |
| `4:3`         | 1024×768  | 2048×1536 | ❌ 최대 픽셀 초과       |
| `3:4`         | 768×1024  | 1536×2048 | ❌ 최대 픽셀 초과       |
| `5:4`         | 1280×1024 | 2560×2048 | ❌ 최대 픽셀 초과       |
| `4:5`         | 1024×1280 | 2048×2560 | ❌ 최대 픽셀 초과       |
| `16:9`        | 1536×864  | 2048×1152 | **3840×2160** |
| `9:16`        | 864×1536  | 1152×2048 | **2160×3840** |
| `2:1`         | 2048×1024 | 2688×1344 | **3840×1920** |
| `1:2`         | 1024×2048 | 1344×2688 | **1920×3840** |
| `21:9`        | 2016×864  | 2688×1152 | **3840×1648** |
| `9:21`        | 864×2016  | 1152×2688 | **1648×3840** |

> `3:2` / `2:3` @ 2K는 실제로 2048×1360 (근사 비율, 오차 \< 0.5%); 4K는 6개 비율만 지원 (총 픽셀이 OpenAI 최대치 초과 조합은 사용 불가).
