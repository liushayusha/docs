## Body

<ParamField body="model" type="string" required>
  모델 이름

  고정 값: `"seedream_v4"`
</ParamField>

<ParamField body="params" type="object" required>
  모델 파라미터 객체

  <Expandable title="params 객체 속성">
    <ParamField body="prompt" type="string" required>
      이미지 생성을 위한 텍스트 설명

      제한 없음
    </ParamField>

    <ParamField body="image_url" type="array">
      참고 이미지 URL 목록, 이미지 변환용

      * 최대 9장
      * 공개 접근 가능한 URL 지원
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="1:1">
      이미지 생성의 가로 세로 비율

      지원 비율:

      * `1:1` - 정사각형(기본값)
      * `16:9` - 가로형 와이드스크린
      * `9:16` - 세로형 화면
      * `4:3` - 가로형 4:3
      * `3:4` - 세로형 3:4
    </ParamField>

    <ParamField body="resolution" type="string" default="1K">
      이미지 해상도

      지원 해상도:

      * `1K` - 기본 해상도(기본값)
      * `2K` - 표준 해상도
      * `4K` - 고해상도
    </ParamField>

    <ParamField body="output_n" type="string" default="1">
      생성할 이미지 수

      지원 값:

      * `1` - 1장 생성(기본값)
      * `2` - 2장 생성
      * `4` - 4장 생성
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  발신자 작업 ID

  사용자가 정의하는 작업 식별자, 필수
</ParamField>
<ParamField body="callback_url" type="string">
  작업 완료 후 콜백 알림 주소

  작업 완료 시 시스템이 해당 주소로 POST 요청을 보내 결과를 알림
</ParamField>

<ParamField body="metadata" type="object">
  사용자 정의 메타데이터

  조회 시 원본 그대로 반환, 비즈니스 식별용으로 사용
</ParamField>

<ResponseField name="out_task_id" type="string">

</ResponseField>
