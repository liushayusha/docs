## Body

<ParamField body="size" type="string" default="1:1">
  이미지 생성 가로세로 비율

  지원하는 가로세로 비율:

  * `1:1` - 정사각형 (기본값)
  * `4:3` - 가로 4:3
  * `3:4` - 세로 3:4
  * `16:9` - 가로 와이드스크린
  * `9:16` - 세로 세로스크린
  * `3:2` - 가로 3:2
  * `2:3` - 세로 2:3
  * `21:9` - 초광폭
  * `9:21` - 초세로스크린
</ParamField>

<ParamField body="n" type="integer">
  생성할 이미지 수량

  값 범위: 1-15 (최소 1장, 최대 15장)

  기본값: 1

  **주의:**

  * 실제 생성 가능한 이미지 수량은 이 매개변수 외에도 입력한 참고 이미지 수량에 영향을 받습니다. 참고 이미지 수 + 최종 생성 이미지 수 ≤ 15장
  * **숫자만 입력해야 합니다(예: `1`), 따옴표 포함 시 오류 발생**

  생성 수량에 따라 선결제 차감됨
</ParamField>

<ParamField body="image_urls" type="array">
  참고 이미지 URL 목록, 이미지 생성 또는 편집에 사용

  두 가지 형식 지원:

  **1. 완전한 이미지 URL 주소**

  * 공개적으로 접근 가능한 이미지 URL (http\:// 또는 https\://)
  * 예시: `https://example.com/image.jpg`

  **2. Base64 인코딩 형식**

  * **전체 Data URI 형식을 반드시 사용해야 합니다**
  * 형식: `data:image/{형식};base64,{base64 데이터}`
  * 지원 이미지 형식: jpeg, png
  * 예시: `data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
  * ⚠️ 주의: `data:image/jpeg;base64,` 접두어 포함 필수

  **제한:**

  * 최대 10장 이미지
  * 이미지 형식: jpeg, png
  * 가로세로 비율(가로/세로) 범위:\[1/3, 3]
  * 가로세로 길이(px) > 14
  * 단일 이미지 크기: 10MB 이하
  * 총 픽셀: 6000×6000 px 이하
</ParamField>

<ParamField body="optimize_prompt_options" type="string" default="standard">
  프롬프트 최적화 모드

  * `standard`: 표준 모드, 생성 콘텐츠 품질이 더 높고 처리 시간 길음
  * `fast`: 빠른 모드, 생성 처리 시간이 더 짧지만 품질은 보통

  기본값: `standard`
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  시리즈 이미지 생성 모드 (Doubao 전용 기능)

  여러 장 이미지 생성을 제어하는 기능:

  * `disabled`: 시리즈 모드 비활성화, 참고 이미지가 여러 장이어도 1장만 생성 (기본값)
  * `auto`: 시리즈 모드 활성화, 여러 장 이미지 생성 가능

  **사용 안내:**

  * ✅ `image_urls` 필수 – 최소 1장의 참고 이미지 필요
  * ✅ `n: 3` 설정 또는 `sequential_image_generation: "auto"` 와 `max_images: 3` 사용
  * ✅ 이렇게 하면 참고 이미지를 기반으로 3장의 서로 다른 이미지 생성
  * ⚠️ `n > 1` 일 경우 자동으로 `auto` 로 설정됨

  **제한:**

  * 참고 이미지가 없는 순수 텍스트 이미지 생성은 다중 이미지 생성 불가, 이는 Doubao API 제한 사항
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  시리즈 이미지 생성 옵션

  `sequential_image_generation` 이 `auto` 로 설정된 경우 사용 가능

  **Properties:**

  * `max_images` (integer): 생성할 이미지 수량 지정, 값 범위: 1-15

  **예시:**

  ```json 
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 3
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  생성된 이미지에 워터마크 추가 여부

  * `true`: 워터마크 추가
  * `false`: 워터마크 미추가 (기본값)
</ParamField>
