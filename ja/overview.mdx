AIReiter は統一された AI コンテンツ生成 API プラットフォームであり、テキスト対話、画像生成、動画生成サービスを提供します。シンプルな REST API 呼び出しを通じて、Claude、GPT、Gemini、Sora2、VEO3 などの主要な AI モデルを利用できます。テキストインターフェースは OpenAI SDK と完全互換で、`baseURL` を直接置き換えて使用可能です。

## API ワークフロー

### 1. 生成タスクの送信

すべての画像および動画生成は非同期処理モードを採用しています。タスクを送信するときに `out_task_id`（発注側タスクID）を渡す必要があり、システムは `out_task_id` を返します：

```bash
curl --request POST \
  --url https://aireiter.com/api/openapi/submit \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "gpt_4o",
    "params": {
      "prompt": "かわいい猫が庭にいる",
      "aspect_ratio": "1:1"
    },
    "out_task_id": "my_task_123456"
  }'
```

### 2. タスクステータスの照会

送信時の `out_task_id` を使ってタスクの進行状況や結果を照会します：

```bash
curl --request POST \
  --url https://aireiter.com/api/openapi/query \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data-raw '{
    "out_task_id": "my_task_123456"
  }'
```

### 3. 生成結果の取得

タスク完了後、レスポンスには生成された画像または動画のURLが含まれます：

```json
{
  "statusCode": 200,
  "message": "",
  "data": {
    "out_task_id": "my_task_123456",
    "status": "completed",
    "output": [
      {
        "url": "https://s1.pxz.ai/upload/image-generator/xxx.jpeg"
      }
    ]
  }
}
```

## コア機能

### 非同期タスク処理

すべての生成リクエストは非同期処理モードを採用し、長時間の待機を回避します：

- タスク提出時に `out_task_id`（発起側のタスクID、ビジネス識別用）を渡す必要があります
- 提出した `out_task_id` をすぐに返します
- `out_task_id` を使用してタスクの状態を照会できます
- webhookコールバック通知をサポート（オプション）
- タスク状態はリアルタイムに更新：`pending` → `processing` → `completed`

### マルチモデル対応

統一されたAPIエンドポイントで、さまざまなテキスト、画像、動画生成モデルをサポート：

- **テキスト対話**：Claude、GPTなどのモデル、OpenAI SDK互換
- **画像生成**：GPT-4o、Gemini、Seedreamなどのモデル
- **動画生成**：Sora2、VEO3、Seedanceなどのモデル
- 統一されたリクエストフォーマットとレスポンス構造
- 柔軟なパラメーター設定

### 透明な価格設定

ポイント制で課金、分かりやすく透明性あり：

- 各タスクの予測消費ポイントを返します
- タスク完了後に実際の消費ポイントを表示
- 成功した生成内容に対してのみ課金
- 隠れた料金なし

### 柔軟な設定

多様な生成パラメーター設定をサポート：

- **画像**：複数のアスペクト比（1:1、3:2、2:3）
- **動画**：複数のアスペクト比（16:9、9:16）、長さ（10秒、15秒）、解像度
- 参照画像のサポート（画像から画像、画像から動画）

  <Accordion title="APIキーはどのように取得しますか？">
    <a href={apiKeyUrl} target="_blank">APIキー管理ページ</a>にアクセスし、ログイン後にAPIキーを作成および管理できます。

  <Accordion title="画像と動画の生成にはどれくらい時間がかかりますか？">
    すべての生成タスクは非同期処理です。画像生成は通常数秒から数十秒かかり、動画生成は長さや解像度によりますが通常1〜5分かかります。タスクを送信するとすぐにout_task_idが返され、ステータス問い合わせAPIを使って進行状況を追跡できます。
  </Accordion>

  <Accordion title="タスクのステータスはどうやって問い合わせますか？">
    タスクステータス問い合わせAPIを使い、自分で設定した`out_task_id`を送信します：

    ```bash
    POST https://aireiter.com/api/openapi/query
    ```

    詳細は[タスクステータス問い合わせドキュメント](/en/api-reference/tasks/status)をご覧ください。
  </Accordion>

  <Accordion title="out_task_idとは何ですか？">
    `out_task_id`は発注側のタスクIDで、タスクを送信するときにカスタム設定します：

    - **必須パラメータ**：タスク送信時に`out_task_id`を必ず渡す必要があります
    - **ビジネス識別**：システムの注文番号やビジネスIDなどを識別子として使えます
    - **柔軟な問い合わせ**：`out_task_id`を使い直接タスクステータスを問い合わせできます
    - **結果関連付け**：問い合わせ結果に`out_task_id`が返され、ビジネスデータと関連付けやすくなります
  </Accordion>

  <Accordion title="料金はどうなっていますか？">
    AIReiterはポイント制の料金体系です：

    - タスク送信時に推定消費ポイント（`estimated_credits`）を返します
    - タスク完了後に実際消費ポイント（`credits_used`）を返します
    - 成功した生成コンテンツにのみ課金し、失敗したタスクは課金しません
    - モデルごとに消費ポイントは異なります
  </Accordion>

  <Accordion title="対応している画像形式と動画形式は何ですか？">
    - **画像**: JPEG、PNGなどの一般的な形式に対応
    - **動画**: 出力はMP4形式
    - **参考画像**: 入力には公開アクセス可能な画像URLをサポート
  </Accordion>
</AccordionGroup>
