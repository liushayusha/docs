APIMart は、画像と動画の生成サービスを提供する統合 AI コンテンツ生成 API プラットフォームです。シンプルな REST API 呼び出しで、GPT-4o、Gemini、Sora2、VEO3 などの様々な主流 AI モデルを使用して、高品質な画像や動画コンテンツを生成できます。

## API ワークフロー

### 1. 生成タスクの送信

すべての画像と動画の生成は非同期処理を採用しています。タスクを送信すると、システムはすぐに `task_id` を返します:

```bash
curl --request POST \
  --url https://pxz-api.xinsulv.com/api/openapi/submit \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "gpt_4o",
    "params": {
      "prompt": "一只可爱的猫咪在花园里",
      "aspect_ratio": "1:1"
    }
  }'
```

### 2. タスクステータスの照会

返された `task_id` を使用してタスクの進捗と結果を照会します:

```bash
curl --request POST \
  --url https://pxz-api.xinsulv.com/api/openapi/query \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data-raw '{
    "task_id": "order_xxxxxxxxxxxx"
  }'
```

### 3. 生成結果の取得

タスクが完了すると、レスポンスに生成された画像または動画の URL が含まれます:

```json
{
  "statusCode": 200,
  "message": "",
  "data": {
    "task_id": "order_xxxxxxxxxxxx",
    "status": "completed",
    "output": [
      {
        "url": "https://s1.pxz.ai/upload/image-generator/xxx.jpeg"
      }
    ]
  }
}
```

## コア機能

### 非同期タスク処理

すべての生成リクエストは非同期処理を採用し、長時間の待機を回避します:

- `task_id` を即座に返却
- タスクステータス API を通じて進捗を照会
- webhook コールバック通知をサポート(オプション)
- タスクステータスのリアルタイム更新: `pending` → `processing` → `completed`

### マルチモデルサポート

複数の画像および動画生成モデルをサポートする統一 API エンドポイント:

- **画像生成**: GPT-4o、Gemini、Seedream など 5 つのモデル
- **動画生成**: Sora2、VEO3 など 4 つのモデル
- 統一されたリクエスト形式とレスポンス構造
- 柔軟なパラメータ設定

### 透明な料金設定

クレジットベースの課金で、透明で明確:

- 各タスクで推定消費クレジットを返却
- タスク完了後に実際の消費クレジットを表示
- 成功した生成コンテンツのみに課金
- 隠れた費用なし

### 柔軟な設定

様々な生成パラメータ設定をサポート:

- **画像**: 複数のアスペクト比(1:1、3:2、2:3)
- **動画**: 複数のアスペクト比(16:9、9:16)、時間(10秒、15秒)、解像度
- 参照画像をサポート(画像から画像、画像から動画)
- カスタムメタデータ

## 認証方法

すべての API リクエストには Bearer Token による認証が必要です:

```
Authorization: Bearer YOUR_API_KEY
```

**API Key の取得:**

[API Key 管理ページ](https://apimart.ai/console/token) にアクセスして API Key を取得してください。

## よくある質問

<AccordionGroup>
  <Accordion title="API Key を取得するにはどうすればよいですか?">
    [API Key 管理ページ](https://apimart.ai/console/token) にアクセスし、ログイン後に API Key を作成および管理できます。
  </Accordion>

  <Accordion title="画像と動画の生成にはどのくらい時間がかかりますか?">
    すべての生成タスクは非同期処理されます。画像生成は通常数秒から数十秒かかり、動画生成は時間と解像度に応じて通常 1-5 分かかります。タスクを送信すると即座に task_id が返され、ステータス照会 API で進捗を追跡できます。
  </Accordion>

  <Accordion title="タスクステータスを照会するにはどうすればよいですか?">
    タスク送信時に返された `task_id` を使用してタスクステータス照会 API を使用します:

    ```bash
    POST https://pxz-api.xinsulv.com/api/openapi/query
    ```

    詳細は [タスクステータス照会ドキュメント](/ja/api-reference/tasks/status) をご覧ください。
  </Accordion>

  <Accordion title="webhook コールバックはサポートされていますか?">
    はい、タスク送信時に `callback_url` パラメータを渡すことができます。タスク完了後、システムはそのアドレスに完全なタスク結果を含む POST リクエストを送信します。
  </Accordion>

  <Accordion title="課金はどのように行われますか?">
    APIMart はクレジット制の課金を採用しています:

    - タスク送信時に推定消費クレジット(`estimated_credits`)を返却
    - タスク完了後に実際の消費クレジット(`credits_used`)を返却
    - 成功した生成コンテンツのみに課金、失敗したタスクは課金されません
    - モデルによって消費クレジットが異なります
  </Accordion>

  <Accordion title="どの画像形式と動画形式がサポートされていますか?">
    - **画像**: JPEG、PNG などの一般的な形式をサポート
    - **動画**: MP4 形式で出力
    - **参照画像**: 公開アクセス可能な画像 URL の入力をサポート
  </Accordion>
</AccordionGroup>
