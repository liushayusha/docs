OpenAI互換の統一されたエンドポイントで、GPT-5、Claude、Geminiをサポートしています。基本URLを`https://api.apimart.ai/v1`に変更するだけで、数分でOpenAIから移行可能—コードを書き直す必要なく、既存のSDKを引き続き使用できます。マルチベンダールーティングで低遅延と高可用性を実現。透明な価格設定、エンタープライズレベルのSLA、グローバルCDNによる高速化。

## API Endpoints

APIMart は、複数の AI モデルとプロバイダーに対応した標準化された画像および動画生成のエンドポイントを提供しています。

### 画像生成 API

GPT-4o Image、Gemini 2.5 Flash Image-preview モデルを使用して画像を生成します。OpenAI スタイルの API を採用しており、シームレスな統合を実現しています。

<Card title="画像生成 API" icon="image" href="/api-reference/images/gpt-4o/generation">
  テキストから画像の生成
</Card>

### 動画生成 API

OpenAI Sora2、Google VEO3 モデルを使用して動画を生成します。非同期タスク管理に対応しており、状態追跡と webhook サポートを備えています。

## Platform Benefits

### 透明な価格設定

利用量に応じた支払いで、サブスクリプションは不要です。すべてのモデルに明確なトークン単価が設定されており、公式プロバイダーよりもお得で、同等の品質と低遅延を維持しています。

* すべてのモデルに明確なトークン単価が設定
* 法人ユーザーは大量割引を享受可能
* 隠れた費用なし
* 実際の利用分のみ支払い

### マルチプロバイダーのルーティングとエンタープライズグレードSLA

複数のLLMプロバイダーを内蔵したインテリジェントルーティングにより、高可用性と低遅延を保証します。プロバイダーに障害が発生した場合は、自動的に予備のプロバイダーにリクエストを切り替え、サービスの中断を防ぎます。

### 99.9%稼働率のSLA

エンタープライズレベルの信頼性と自動フェイルオーバーを備えています。マルチプロバイダーのルーティングにより、常に最低遅延でリクエストが処理されます。

### グローバルCDNアクセラレーション

世界中のエッジノードにより最低遅延を提供します。最適化されたルーティングで応答時間を短縮し、世界中のユーザー体験を向上させます。

### レート制限管理

プロバイダー間のレート制限を自動で処理します。インテリジェントなリクエスト配分によりスロットリングを防ぎ、スムーズな運用を保証します。

### リアルタイムステータス

エンドポイントのヘルスとパフォーマンスを監視します。画像や動画生成の非同期タスクの進捗を追跡し、Webhookで通知します。[ステータスを見る →](/en/api-reference/tasks/status)

## Supported Models

統一されたAPIエンドポイントを通じて、先進的な画像および動画生成AIモデルにアクセスします。

### Image Creation Models

テキストプロンプトに基づく画像生成：

* **GPT-4o Image**：OpenAI画像生成モデル
* **Gemini 2.5 Flash Image-preview**：Googleの高性能画像モデル

### Video Creation Models

非同期タスク追跡とwebhookコールバックに対応した動画生成：

* **OpenAI Sora2**：OpenAI動画生成モデル
* **Google VEO3**：Google動画生成モデル
* **doubao-seedance-1-0-pro**：高速で高品質な動画生成
* **MiniMax-Hailuo-02**：画像から動画への変換をサポートする効率的な動画生成

## FAQ

<AccordionGroup>
  <Accordion title="画像と動画の生成はどのように機能しますか？">
    画像生成は同期的で、すぐに結果を返すことができます。動画生成は非同期で、リクエストを送信すると task_id が返され、その後ステータスのエンドポイントをポーリングして進捗を確認し、最終的な動画の URL を取得します。
  </Accordion>

  <Accordion title="SLAには何が含まれていますか？">
    当社のエンタープライズSLAには、99.9%の稼働率保証、グローバルCDNアクセラレーション、マルチベンダールーティングと自動フェイルオーバー、リアルタイム状態監視が含まれます。非同期タスクはレート制限管理とWebhookをサポートしています。
  </Accordion>

  <Accordion title="料金はどのように計算されますか？">
    APIMartは画像・動画生成に対し、透明性のあるリクエスト単位の課金を提供しています。サブスクリプション不要の従量制で、エンタープライズユーザーにはまとめて割引があり、隠れた料金はありません。成功した生成コンテンツのみ課金されます。
  </Accordion>

  <Accordion title="遅延とパフォーマンスはどうですか？">
    グローバルCDNアクセラレーションとスマートなマルチベンダールーティングにより低遅延を保証します。画像生成は通常即時ですが、動画生成時間はモデルによって異なり（40～300秒）、リアルタイム状態監視により非同期タスクの進捗を追跡できます。
  </Accordion>
</AccordionGroup>
