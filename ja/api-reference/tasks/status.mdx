---
title: 'タスクステータスの取得'
description: |
    - 非同期タスクの実行状況と結果の照会
    - リアルタイムステータス更新と進行状況の追跡
    - タスク完了時に生成結果を取得
    - 多言語対応（zh/en/ko/ja）
api: 'GET https://api.apimart.ai/v1/tasks/{task_id}'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearer Tokenによる認証が必要です

  APIキーの取得：

  [API Key 管理ページ](https://apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  利用時はリクエストヘッダーに以下を追加します：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Path parameters

<ParamField path="task_id" type="string" required>
  生成APIが返すタスクID
</ParamField>

## Query parameters

<ParamField query="language" type="string">
  返却されるコンテンツの言語で、以下の値をサポートします：

  * `zh` - 中国語
  * `en` - 英語
  * `ko` - 韓国語
  * `ja` - 日本語

  デフォルトは英語を返却します
</ParamField>

## Response

<ResponseField name="id" type="string">
  タスクの一意識別子
</ResponseField>

<ResponseField name="status" type="string">
  タスクの状態値：

  * `pending` - 処理待ちのキュー
  * `processing` - 処理中
  * `completed` - 成功完了
  * `failed` - 失敗
  * `cancelled` - ユーザーによるキャンセル
</ResponseField>

<ResponseField name="progress" type="integer">
  タスクの進捗率（0～100）
</ResponseField>

<ResponseField name="result" type="object">
  タスク結果。状態が `completed` の時のみ返されます

  <Expandable title="Properties">
    <ResponseField name="images" type="array">
      生成された画像オブジェクトの配列（画像生成タスク）
    </ResponseField>

    <ResponseField name="videos" type="array">
      生成された動画オブジェクトの配列（動画生成タスク）
    </ResponseField>

    <ResponseField name="thumbnail_url" type="string">
      動画のサムネイルURL（動画生成タスク）

      <Note>
        **重要:** このフィールドは、タスク作成時にサムネイル生成が明示的に要求された場合のみ返されます。サムネイル生成を指定しなかった場合、このフィールドはレスポンスに含まれません。
      </Note>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created" type="integer">
  タスク作成のタイムスタンプ
</ResponseField>

<ResponseField name="completed" type="integer">
  タスク完了のタイムスタンプ（完了時のみ存在）
</ResponseField>

<ResponseField name="estimated_time" type="integer">
  予想完了時間（秒）
</ResponseField>

<ResponseField name="actual_time" type="integer">
  実際の完了時間（秒）（完了時のみ存在）
</ResponseField>

<ResponseField name="error" type="object">
  エラー詳細（状態が `failed` の時のみ存在）

  <Expandable title="Properties">
    <ResponseField name="code" type="integer">
      エラーコード
    </ResponseField>

    <ResponseField name="message" type="string">
      エラーメッセージ
    </ResponseField>

    <ResponseField name="type" type="string">
      エラータイプ
    </ResponseField>
  </Expandable>
</ResponseField>
