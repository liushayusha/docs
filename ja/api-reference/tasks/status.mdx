## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearer Tokenを使った認証が必要です

  APIキーの取得：

  <a href={apiKeyUrl} target="_blank">APIキー管理ページ</a>にアクセスしてAPIキーを取得してください

  利用時はリクエストヘッダーに以下を追加します：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="task_id" type="string">
  生成APIが返すタスクID

  out_task_idと二者択一で、いずれか一方の提供が必須
</ParamField>

<ParamField body="out_task_id" type="string">
  発信側タスクID

  task_idと二者択一で、いずれか一方の提供が必須
</ParamField>

## レスポンス

<ResponseField name="task_id" type="string">
  タスクID
</ResponseField>

<ResponseField name="status" type="string">
  タスクステータス：

  * `pending` - 処理待ちキュー
  * `processing` - 処理中
  * `completed` - 成功完了
  * `failed` - 失敗
</ResponseField>

<ResponseField name="input" type="object">
  タスク入力（オプション、特定の場合のみ返却）

  タスク提出時の完全なパラメータ情報を含む
</ResponseField>

<ResponseField name="output" type="array">
  タスク出力（タスク完了後に返却）

  生成結果の配列を含み、各要素は生成されたリソースを指す `url` フィールドを含む
</ResponseField>

<ResponseField name="error" type="object">
  エラー情報（タスク失敗時に返却）

  <Expandable title="プロパティ">
    <ResponseField name="code" type="string">
      エラーコード
    </ResponseField>

    <ResponseField name="message" type="string">
      エラーメッセージ
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="credits_used" type="number">
  使用したクレジット
</ResponseField>

<ResponseField name="created_at" type="string">
  作成日時（ISO 8601形式）
</ResponseField>

<ResponseField name="completed_at" type="string">
  完了日時（タスク完了後に返却、ISO 8601形式）
</ResponseField>

  ```c C
  #include <stdio.h>
  #include <curl/curl.h>

  int main(void) {
      CURL *curl;
      CURLcode res;

      curl_global_init(CURL_GLOBAL_DEFAULT);
      curl = curl_easy_init();

      if(curl) {
          const char *url = "https://aireiter.com/api/openapi/query";

          struct curl_slist *headers = NULL;
          headers = curl_slist_append(headers, "Authorization: Bearer <token>");
          headers = curl_slist_append(headers, "Content-Type: application/json");

          const char *data = "{\"task_id\":\"order_lwU2d-VkUq9-tlCCmrdYu\"}";

          curl_easy_setopt(curl, CURLOPT_URL, url);
          curl_easy_setopt(curl, CURLOPT_HTTPHEADER, headers);
          curl_easy_setopt(curl, CURLOPT_POSTFIELDS, data);

          res = curl_easy_perform(curl);

          if(res != CURLE_OK) {
              fprintf(stderr, "curl_easy_perform() failed: %s\n",
                      curl_easy_strerror(res));
          }

          curl_slist_free_all(headers);
          curl_easy_cleanup(curl);
      }

      curl_global_cleanup();
      return 0;
  }
  ```

  ```objectivec Objective-C
  #import <Foundation/Foundation.h>

  int main(int argc, const char * argv[]) {
      @autoreleasepool {
          NSURL *url = [NSURL URLWithString:@"https://aireiter.com/api/openapi/query"];

          NSMutableURLRequest *request = [NSMutableURLRequest requestWithURL:url];
          [request setHTTPMethod:@"POST"];
          [request setValue:@"Bearer <token>" forHTTPHeaderField:@"Authorization"];
          [request setValue:@"application/json" forHTTPHeaderField:@"Content-Type"];

          NSDictionary *payload = @{
              @"task_id": @"order_lwU2d-VkUq9-tlCCmrdYu"
          };

          NSData *jsonData = [NSJSONSerialization dataWithJSONObject:payload options:0 error:nil];
          [request setHTTPBody:jsonData];

          NSURLSessionDataTask *task = [[NSURLSession sharedSession]
              dataTaskWithRequest:request
              completionHandler:^(NSData *data, NSURLResponse *response, NSError *error) {
                  if (error) {
                      NSLog(@"Error: %@", error);
                      return;
                  }
                  NSString *result = [[NSString alloc] initWithData:data
                                                          encoding:NSUTF8StringEncoding];
                  NSLog(@"%@", result);
              }];

          [task resume];
          [[NSRunLoop mainRunLoop] run];
      }
      return 0;
  }
  ```
