title: 'Seedream V4 画像生成'
description: |
    - 非同期処理モードで、後続のクエリに使用するタスクIDを返します
    - テキストから画像、画像から画像への多様な生成モードをサポート
api: 'POST https://pxz-api.xinsulv.com/api/openapi/submit'

import { apiKeyUrl } from '/snippets/_variables.mdx';

## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearer Tokenによる認証が必要です。

  API Keyの取得：

  <a href={apiKeyUrl} target="_blank">API Key管理ページ</a>にアクセスしてAPI Keyを取得してください。

  使用時はリクエストヘッダーに以下を追加します：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" required>
  モデル名

  固定値：`"seedream_v4"`
</ParamField>

<ParamField body="params" type="object" required>
  モデルパラメーターオブジェクト

  <Expandable title="params オブジェクトのプロパティ">
    <ParamField body="prompt" type="string" required>
      画像生成のテキスト説明

      制限なし
    </ParamField>

    <ParamField body="image_url" type="array">
      画像生成の参考用画像のURLリスト

      * 最大9枚の画像
      * 公開アクセス可能なURLをサポート
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="1:1">
      画像生成のアスペクト比

      サポートされるアスペクト比：

      * `1:1` - 正方形（デフォルト）
      * `16:9` - 横長ワイドスクリーン
      * `9:16` - 縦長スクリーン
      * `4:3` - 横長4:3
      * `3:4` - 縦長3:4
    </ParamField>

    <ParamField body="resolution" type="string" default="1K">
      画像解像度

      サポートされる解像度：

      * `1K` - ベース解像度（デフォルト）
      * `2K` - 標準解像度
      * `4K` - 高解像度
    </ParamField>

    <ParamField body="output_n" type="string" default="1">
      生成する画像の枚数

      サポートされる値：

      * `1` - 1枚生成（デフォルト）
      * `2` - 2枚生成
      * `4` - 4枚生成
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  発起者タスクID

  ユーザーが定義するタスク識別子、必須
</ParamField>
<ParamField body="callback_url" type="string">
  タスク完了後のコールバック通知先URL

  タスク完了後、システムはこのURLにPOSTリクエストで結果を通知します
</ParamField>

<ParamField body="metadata" type="object">
  カスタムメタデータ

  クエリ結果時にそのまま返され、ビジネス識別に利用可能
</ParamField>

## Response

<ResponseField name="task_id" type="string">
  タスクID、結果の照会に使用します
</ResponseField>

<ResponseField name="out_task_id" type="string">
  発信者側のタスクID、結果の照会に使用できます
</ResponseField>
<ResponseField name="status" type="string">
  初期状態、固定で `"pending"`
</ResponseField>

<ResponseField name="estimated_credits" type="number">
  推定消費クレジット
</ResponseField>

<ResponseField name="created_at" type="string">
  作成日時（ISO形式）
</ResponseField>
