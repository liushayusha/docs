---
title: 'Seedream-4.0 画像生成'
description: |
    - 非同期処理モードで、タスクIDを返し、後続のクエリに使用できます
    - テキストから画像、画像から画像、画像編集など多様な生成モードに対応
    - 生成された画像のリンクは有効期限が24時間のため、速やかに保存してください
api: 'POST https://api.apimart.ai/v1/images/generations'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearerトークンによる認証が必要です

  APIキーの取得方法：

  [APIキー管理ページ](https://apimart.ai/console/token) にアクセスしてAPIキーを取得してください

  使用時はリクエストヘッダーに以下を追加してください：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="doubao-seedance-4-0" required>
  画像生成モデル名

  Example: `"doubao-seedance-4-0"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のテキスト説明

  最大 1000 文字
</ParamField>

<ParamField body="size" type="string" default="1:1">
  画像生成のアスペクト比

  対応アスペクト比：

  * `1:1` - 正方形（デフォルト）
  * `4:3` - 横長 4:3
  * `3:4` - 縦長 3:4
  * `16:9` - 横長ワイドスクリーン
  * `9:16` - 縦長縦画面
  * `3:2` - 横長 3:2
  * `2:3` - 縦長 2:3
  * `21:9` - 超ワイドスクリーン
  * `9:21` - 超縦長画面
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  画像解像度

  対応解像度：

  * `1K` - 基本解像度
  * `2K` - 標準解像度（デフォルト）
  * `4K` - 高解像度

  **解像度の参考サイズ：**

  | 解像度 | 1:1 サイズ    | 16:9 サイズ   |
  | --- | --------- | --------- |
  | 1K  | 1024x1024 | 1280x720  |
  | 2K  | 2048x2048 | 2560x1440 |
  | 4K  | 4096x4096 | 5404x3040 |

  <Expandable title="完全ピクセル対応表">
    **1K 解像度**

    | 比率   | ピクセル        |
    | ---- | --------- |
    | 1:1  | 1024x1024 |

    | 16:9 | 1280x720  |
    | 9:16 | 720x1280  |
    | 3:2  | 1248x832  |

<ParamField body="n" type="integer">
  生成する画像の枚数

  範囲：1-15（最少 1 枚、最大 15 枚）

  デフォルト値：1

  **注意：**

  * 実際に生成できる画像枚数は、このパラメータの影響以外に、入力した参考画像の数にも依存します。参考画像の数 + 最終生成画像の数 ≤ 15 枚
  * **数字のみを入力してください（例: `1`）、引用符を付けるとエラーになります**

  生成枚数に応じて事前課金が行われます
</ParamField>

<ParamField body="image_urls" type="array">
  参考画像の URL リスト（画像から画像生成や画像編集用）

  2 つの形式に対応：

  **1. 完全な画像 URL アドレス**

  * 公開アクセス可能な画像 URL（http\:// または https\://）
  * 例：`https://example.com/image.jpg`

  **2. Base64 エンコード形式**

  * **完全な Data URI 形式を使用する必要があります**
  * 形式：`data:image/{フォーマット};base64,{base64データ}`
  * 対応画像フォーマット：jpeg、png
  * 例：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
  * ⚠️ 注意：`data:image/jpeg;base64,` のプレフィックス部分を必須含む

  **制限：**

  * 最大 10 枚の画像
  * 画像フォーマット：jpeg、png
  * アスペクト比（幅/高さ）：\[1/3, 3]
  * 幅または高さ（px）> 14
  * 1 枚あたりのサイズ：最大 10 MB
  * 総ピクセル数：最大 6000×6000 px
</ParamField>

<ParamField body="optimize_prompt_options" type="string" default="standard">
  プロンプト最適化モード

  * `standard`：標準モード、生成される内容の品質が高く、時間がかかる
  * `fast`：高速モード、生成時間が短く品質は普通

  デフォルト値：`standard`
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  連続画像生成モード（doubao 専用機能）

  複数画像生成の制御に使用：

  * `disabled`：連続画像モードを無効化、参考画像が複数あっても 1 枚だけ生成（デフォルト）
  * `auto`：連続画像モードを有効化、複数枚生成可能

  **使用説明：**

  * ✅ `image_urls`を必ず提供してください - 少なくとも1枚の参考画像が必要
  * ✅ `n: 3` を設定するか、または `sequential_image_generation: "auto"` と `max_images: 3` を使用
  * ✅ これで参考画像を基に3枚の異なる画像を生成します
  * ⚠️ `n > 1` の場合は自動的に `auto` に設定されます

  **制限：**

  * 参考画像がないテキストからの画像生成のみの場合、複数枚は生成できません。これは doubao API の制限です
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  連続画像生成オプション

  `sequential_image_generation` が `auto` の時に利用可能

  **プロパティ：**

  * `max_images` (integer): 生成する画像の枚数、範囲：1-15

  **例：**

  ```json 
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 3
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  生成された画像にウォーターマークを追加するかどうか

  * `true`：ウォーターマークを追加
  * `false`：ウォーターマーク無し（デフォルト）
</ParamField>
