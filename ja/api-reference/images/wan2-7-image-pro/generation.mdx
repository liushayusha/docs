---
title: 'Wan2.7-Image-Pro 画像生成'
description: |
    - 非同期処理モード、タスクIDを返し、後続のクエリに使用可能
    - テキストから画像、画像から画像の2つの生成モードをサポート
    - 複数の画像アスペクト比をサポートし、aspect_ratio フィールドで指定（デフォルトは auto で自動適応）
    - 1K / 2K / 4K 解像度レベルを選択可能
    - 一度に複数枚の画像生成をサポートし、output_n フィールドで制御
    - 提出されたプロンプトはプラットフォームの敏感語・安全審査を通過し、違反内容は直接拒否される
api: 'POST https://aireiter.com/api/openapi/submit'

## Body

<ParamField body="model" type="string" required>
  モデル名

  固定値：`"wan2_7_image_pro"`
</ParamField>

<ParamField body="params" type="object" required>
  モデルパラメータオブジェクト

  <Expandable title="params オブジェクトのプロパティ">
    <ParamField body="prompt" type="string" required>
      画像生成のテキスト説明

      * 中国語と英語をサポートし、詳細な記述を推奨
      * 提出前にプラットフォームの敏感語/安全監査を通過し、違反内容があれば直接エラーを返します
    </ParamField>

    <ParamField body="image_url" type="string[]">
      参照画像のURLリスト。入力すると画像から画像への生成モードになります

      * 以下の2つの形式をサポートします：
        1. 公開アクセス可能なURL
        2. Base64エンコード形式
           * 完全なData URI形式である必要があります：`data:image/{形式};base64,{base64データ}`
           * 対応画像フォーマット：jpeg、png、webp
           * 例：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
           * ⚠️ 注意：`data:image/jpeg;base64,`のプレフィックスが必須です
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="auto">
      画像生成のアスペクト比

      次の値をサポートします：

      * `auto` - 最適な比率を自動選択（デフォルト）
      * `1:1` - 正方形
      * `16:9` / `9:16` - ワイドスクリーン横 / 縦
      * `4:3` / `3:4` - 標準スクリーン横 / 縦
      * `3:2` / `2:3` - クラシック横 / 縦

      <Warning>
        比率の文字列または `auto` のみサポート；ピクセルサイズ（例：`1024x1024`）はエラーになります
      </Warning>
    </ParamField>

    <ParamField body="resolution" type="string" default="2K">
      画像解像度レベル

      実際の生成画像の鮮明さを制御します：

      * `1K` - 基本解像度、コストを抑えた日常利用向け
      * `2K` - 標準解像度、ポスター/高画質ニーズに適合（デフォルト）
      * `4K` - 高解像度
    </ParamField>

    <ParamField body="output_n" type="string" default="1">
      生成画像の枚数

      サポートされる値：

      * `1` - 1枚生成（デフォルト）
      * `2` - 2枚生成
      * `4` - 4枚生成
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="out_task_id" type="string" required>
  発信者タスクID

  ユーザーがカスタマイズするタスク識別子、必須
</ParamField>
