---
title: 'MiniMax-Hailuo-02 ビデオ生成'
description: |
    - 非同期処理モード、後続のクエリで使用するタスクIDを返却
    - テキストからビデオ、画像からビデオ（最初のフレーム/最後のフレーム）をサポート
    - 6秒および10秒の長さに対応し、複数の解像度を選択可能
    - プロンプトの自動最適化とウォーターマーク制御をサポート
api: 'POST https://api.apimart.ai/v1/videos/generations'
---
## 認証

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントでBearer Tokenによる認証が必要です

  APIキーの取得方法：

  [APIキー管理ページ](https://apimart.ai/console/token) にアクセスしてAPIキーを取得してください

  使用時はリクエストヘッダーに以下を追加します：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## リクエストパラメータ

<ParamField body="model" type="string" required>
  ビデオ生成モデルの名前

  固定値：`MiniMax-Hailuo-02`
</ParamField>

<ParamField body="prompt" type="string" required>
  ビデオ内容の説明

  シーン、動作、スタイルなどを詳細に記述することを推奨します。より良い生成結果を得るためです。

  例：`"かわいい猫が草原を走っている"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  ビデオの長さ（秒）

  選択可能な値：

  * `5` - 5秒のビデオ
  * `10` - 10秒のビデオ

  デフォルト値：`5`

  <Warning>
    **1080p の制限**：1080p 解像度使用時、5秒の長さのみ対応
  </Warning>
</ParamField>

<ParamField body="metadata" type="object">
  ビデオ生成の高度な設定オプション

  <Expandable title="metadata フィールド説明">
    <ParamField body="metadata.resolution" type="string" default="768p">
      ビデオ解像度

      選択可能な値：

      * `512p` - 標準画質
      * `768p` - 高画質
      * `1080p` - フルハイビジョン（6秒の長さのみ対応）

      デフォルト値：`768p`
    </ParamField>

    <ParamField body="metadata.prompt_optimizer" type="boolean" default="true">
      プロンプトの自動最適化の可否

      有効にすると、システムがより良い生成結果を得るためにプロンプトを自動最適化します。

      デフォルト値：`true`
    </ParamField>

    <ParamField body="metadata.fast_pretreatment" type="boolean" default="false">
      プロンプト最適化の時間短縮の可否

      有効にすると処理速度が速くなりますが、最適化効果がわずかに低下する可能性があります。

      デフォルト値：`false`
    </ParamField>

    <ParamField body="metadata.watermark" type="boolean" default="false">
      透かしを追加するかどうか

      デフォルト値：`false`
    </ParamField>

    <ParamField body="metadata.first_frame_image" type="string">
      ビデオの開始フレーム画像

      2つの形式をサポート：

      * **パブリックURL**：`https://example.com/start.jpg`
      * **Base64 エンコード**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      ビデオの開始画面を指定するために使用します。
    </ParamField>

    <ParamField body="metadata.last_frame_image" type="string">
      ビデオの終了フレーム画像

      2つの形式をサポート：

      * **パブリックURL**：`https://example.com/end.jpg`
      * **Base64 エンコード**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      ビデオの終了画面を指定するために使用します。
    </ParamField>
  </Expandable>
</ParamField>

## パラメータ制限

| 制限項目        | 説明                                                   |
| --------- | -------------------------------------------------- |
| 時間長        | 5秒または10秒のみ対応                                           |
| 1080p 解像度 | 5秒の時間長のみ対応                                            |
| 画像形式      | 公衆URLまたはBase64エンコード(`data:image/jpeg;base64,...`)をサポート                 |

## 分辨率と時間の組み合わせ

| 解像度   | 対応時間  | 備考     |
| ----- | ------ | ------ |
| 512p  | 5秒、10秒 | 全て対応   |
| 768p  | 5秒、10秒 | 全て対応   |
| 1080p | 5秒     | 10秒は非対応 |

<ResponseField name="code" type="integer">

    <ResponseField name="status" type="string">

    <ResponseField name="task_id" type="string">
      タスクの一意識別子。タスクのステータスや結果確認に使用
    </ResponseField>
  </Expandable>
</ResponseField>

## 使用事例

### 事例 1：テキスト動画を素早く生成

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "可愛い猫が草原を走っていて、日差しは明るい"
}
```

### 事例 2：高品質の1080p動画を生成

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "都市の夜景、ネオンライトが煌めき、車が行き交う",
  "duration": 5,
  "metadata": {
    "resolution": "1080p",
    "prompt_optimizer": true,
    "watermark": false
  }
}
```

### 事例 3：最初のフレーム画像を基に動画を生成

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "人物がゆっくりと振り向き、微笑んでいる",
  "duration": 5,
  "metadata": {
    "resolution": "768p",
    "first_frame_image": "https://example.com/portrait.jpg"
  }
}
```

### 事例 4：最初と最後のフレームでトランジションを制御する動画

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "映像が昼間から徐々に夜に移り変わり、空がグラデーションになる",
  "duration": 10,
  "metadata": {
    "resolution": "768p",
    "first_frame_image": "https://example.com/day.jpg",
    "last_frame_image": "https://example.com/night.jpg",
    "prompt_optimizer": true
  }
}
```

### 事例 5：迅速な前処理モード

```json
  "duration": 5,
  "metadata": {

    "fast_pretreatment": true
  }
}
```

<Note>
  **タスク結果の照会**

  動画生成は非同期タスクで、提出後に `task_id` が返されます。[タスク状態の取得](/cn/api-reference/tasks/status) エンドポイントで生成の進行状況と結果を照会してください。
</Note>
