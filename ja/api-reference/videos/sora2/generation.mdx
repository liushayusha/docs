---
title: 'Sora2 ビデオ生成'
description: |
    - 非同期処理モード、後続のクエリ用にタスクIDを返す
    - テキストからビデオ生成、画像からビデオ生成などの生成モードをサポート
api: 'POST https://pxz-api.xinsulv.com/api/openapi/submit'
---
## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearer Tokenによる認証が必要です

  APIキーの取得方法：

  [APIキー管理ページ](https://apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  使用時はリクエストヘッダーに以下を追加します：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" required>
  モデル名

  固定値：`"sora2"`
</ParamField>

<ParamField body="params" type="object" required>
  モデルパラメーターオブジェクト

  <Expandable title="params オブジェクトのプロパティ">
    <ParamField body="prompt" type="string" required>
      動画生成のテキスト説明

      制限なし
    </ParamField>

    <ParamField body="image_url" type="string">
      参考画像のURL、動画生成（最初のフレーム）用

      * 単一画像のみ対応
      * 公開アクセス可能なURLをサポート
    </ParamField>

    <ParamField body="aspect_ratio" type="string" default="16:9">
      動画のアスペクト比

      対応アスペクト比：

      * `16:9` - 横向き（デフォルト）
      * `9:16` - 縦向き
    </ParamField>

    <ParamField body="video_length" type="string" default="10">
      動画の長さ（秒）

      対応長さ：

      * `10` - 10秒（デフォルト）
      * `15` - 15秒
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="callback_url" type="string">
  タスク完了後のコールバック通知先URL

  タスク完了後、システムはこのURLにPOSTリクエストを送信して結果を通知します
</ParamField>

<ParamField body="metadata" type="object">
  カスタムメタデータ

  クエリ結果時にそのまま返され、ビジネス識別に利用可能
</ParamField>

## Response

<ResponseField name="task_id" type="string">
  タスクID、結果を照会するために使用されます
</ResponseField>

<ResponseField name="status" type="string">
  初期状態、固定で `"pending"`
</ResponseField>

<ResponseField name="estimated_credits" type="number">
  予想消費クレジット
</ResponseField>

<ResponseField name="created_at" type="string">
  作成日時（ISO形式）
</ResponseField>
