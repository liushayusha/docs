---
title: 'Sora2 キャラクタークエリ（キャラクター管理）'
description: |
    - キャラクター作成タスクの状態と結果をクエリします
    - 作成済みキャラクターの詳細情報を取得します
    - 返されるキャラクターIDは後続のビデオ生成に利用可能です
api: 'GET https://api.apimart.ai/v1/characters_tasks/{task_id}'
---
## パスパラメータ

<ParamField path="task_id" type="string" required>
  ロール作成タスクID

  これはロール作成APIから返されるタスクIDです

  例：`"task_01KBZ0TZZ7C7M3WS1TJM0PH8J4"`
</ParamField>

## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントは Bearer Token を使用した認証が必要です

  API Key の取得：

  [API Key 管理ページ](https://apimart.ai/console/token) にアクセスして、API Key を取得してください

  使用時はリクエストヘッダーに以下を追加してください：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Response

<ResponseField name="code" type="integer">
  レスポンスステータスコード、成功時は 200
</ResponseField>

<ResponseField name="data" type="object">
  返却データオブジェクト

  <Expandable title="フィールド説明">
    <ResponseField name="id" type="string">
      タスクの一意識別子

      例：`"task_01KC0JZCMTMQ70D68XTM56Q5D0"`
    </ResponseField>

    <ResponseField name="status" type="string">
      タスクステータス

      可能な値：

      * `pending` - 処理待ち
      * `processing` - 処理中
      * `completed` - 完了済み
      * `failed` - 失敗

      例：`"completed"`
    </ResponseField>

    <ResponseField name="progress" type="integer">
      タスクの進捗度合いのパーセンテージ (0-100)

      例：`100`
    </ResponseField>

    <ResponseField name="created" type="integer">
      タスク作成タイムスタンプ（Unixタイムスタンプ）

      例：`1765251461`
    </ResponseField>

    <ResponseField name="completed" type="integer">
      タスク完了タイムスタンプ（Unixタイムスタンプ）

      例：`1765251507`
    </ResponseField>

    <ResponseField name="estimated_time" type="integer">
      予想処理時間（秒）

      例：`30`
    </ResponseField>

    <ResponseField name="actual_time" type="integer">
      実際の処理時間（秒）

      例：`46`
    </ResponseField>

    <ResponseField name="result" type="object">
      タスク結果

      <Expandable title="result フィールド">
        <ResponseField name="characters" type="array">
          キャラクター配列

          <Expandable title="キャラクターオブジェクト">
            <ResponseField name="id" type="string">
              キャラクターの一意識別子

              この ID は、動画生成時に `character_url` パラメータを使ってキャラクターを参照できます

              例：`"ch_6937998961208191a45ef08447a554df"`
            </ResponseField>

            <ResponseField name="display_name" type="string">
              キャラクター表示名

              例：`"Turbo Whiskers"`
            </ResponseField>

            <ResponseField name="profile_picture_url" type="string">
              キャラクターのプロフィール写真のURL

              例：`"https://upload.apimart.ai/f/image/...character_task_xxx.jpg"`
            </ResponseField>

            <ResponseField name="username" type="string">
              キャラクターのユーザー名。プロンプト内で `@username` を使ってキャラクターを参照可能

              例：`"duksvfkf.turbo_whis"`
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
