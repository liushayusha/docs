## Authorization

<ParamField header="Authorization" type="string" required>
  所有 API 请求都需要 Bearer Token 认证

  获取 API 密钥：

  访问 [API Key Management Page](https://apimart.ai/console/token) 获取您的 API 密钥

  添加到请求头：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Request Parameters

<ParamField body="model" type="string" required>
  视频生成模型名称

  固定值：`MiniMax-Hailuo-02`
</ParamField>

<ParamField body="prompt" type="string" required>
  视频内容描述

  详细描述场景、动作、风格等，以获得更好的生成效果

  示例：`"一只可爱的猫在草地上奔跑"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长（秒）

  选项：

  * `5` - 5秒视频
  * `10` - 10秒视频

  默认值：`5`

  <Warning>
    **1080p 限制**：使用1080p分辨率时，仅支持5秒时长
  </Warning>
</ParamField>

<ParamField body="metadata" type="object">
  视频生成的高级配置选项

  <Expandable title="metadata fields">
    <ParamField body="metadata.resolution" type="string" default="768p">
      视频分辨率

      选项：

      * `512p` - 标清
      * `768p` - 高清
      * `1080p` - 全高清（仅支持5秒时长）

      默认值：`768p`
    </ParamField>

    <ParamField body="metadata.prompt_optimizer" type="boolean" default="true">
      是否自动优化提示词

      启用后，系统会自动优化你的提示词以获得更好生成效果

      默认值：`true`
    </ParamField>

    <ParamField body="metadata.fast_pretreatment" type="boolean" default="false">
      是否减少提示词优化时间

      启用后可以加快处理速度，但可能会稍微影响优化质量

      默认值：`false`
    </ParamField>

    <ParamField body="metadata.watermark" type="boolean" default="false">
      是否添加水印

      默认值：`false`
    </ParamField>

    <ParamField body="metadata.first_frame_image" type="string">
      视频首帧图像

      支持两种格式：

      * **公开 URL**：`https://example.com/start.jpg`
      * **Base64 编码**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      用于指定视频的起始画面
    </ParamField>

    <ParamField body="metadata.last_frame_image" type="string">
      视频末帧图像

      支持两种格式：

      * **公开 URL**：`https://example.com/end.jpg`
      * **Base64 编码**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

      用于指定视频的结束画面
    </ParamField>
  </Expandable>
</ParamField>

## Parameter Limitations

| Limitation       | Description                                                          |
| ---------------- | -------------------------------------------------------------------- |
| Duration         | 仅支持5秒或10秒                                                     |
| 1080p Resolution | 仅支持5秒的时长                                                    |
| Image Format     | 支持公共URL或Base64编码（`data:image/jpeg;base64,...`）          |

## Resolution and Duration Combinations

| Resolution | Supported Duration | Notes             |
| ---------- | ------------------ | ----------------- |
| 512p       | 5s, 10s            | 全部支持          |
| 768p       | 5s, 10s            | 全部支持          |
| 1080p      | 5s                 | 不支持10秒        |

    <ResponseField name="task_id" type="string">

  "prompt": "一只可爱的猫在阳光明媚的草地上跑"
}

<Note>

</Note>
