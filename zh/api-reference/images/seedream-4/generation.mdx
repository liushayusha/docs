## Authorizations

<ParamField header="Authorization" type="string" required>
  所有API端点均需Bearer Token认证

  获取您的API密钥：

  访问[API密钥管理页面](https://apimart.ai/console/token)以获取您的API密钥

  将其添加到请求头中：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="doubao-seedance-4-0" required>
  图像生成模型名称

  示例：`"doubao-seedance-4-0"`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述

  最多1000个字符
</ParamField>

<ParamField body="size" type="string" default="1:1">
  图像长宽比

  支持的长宽比：

  * `1:1` - 正方形（默认）
  * `4:3` - 横向4:3
  * `3:4` - 纵向3:4
  * `16:9` - 横向宽屏
  * `9:16` - 纵向竖屏
  * `3:2` - 横向3:2
  * `2:3` - 纵向2:3
  * `21:9` - 超宽屏
  * `9:21` - 超高屏
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  图像分辨率

  支持的分辨率：

  * `1K` - 基础分辨率
  * `2K` - 标准分辨率（默认）
  * `4K` - 高清分辨率

  **分辨率参考尺寸：**

  | 分辨率  | 1:1 尺寸   | 16:9 尺寸  |
  | ------- | ---------- | ---------- |
  | 1K      | 1024x1024  | 1280x720   |
  | 2K      | 2048x2048  | 2560x1440  |
  | 4K      | 4096x4096  | 5404x3040  |

  <Expandable title="完整像素映射表">
    **1K 分辨率**

    | 比例  | 像素      |
    | ----- | --------- |
    | 1:1   | 1024x1024 |
    | 4:3   | 1152x864  |
    | 3:4   | 864x1152  |
    | 16:9  | 1280x720  |
    | 9:16  | 720x1280  |
    | 3:2   | 1248x832  |
    | 2:3   | 832x1248  |
    | 21:9  | 1512x648  |
    | 9:21  | 648x1512  |

    **2K 分辨率**

    | 比例  | 像素      |
    | ----- | --------- |
    | 1:1   | 2048x2048 |
    | 4:3   | 2304x1728 |
    | 3:4   | 1728x2304 |
    | 16:9  | 2560x1440 |
    | 9:16  | 1440x2560 |
    | 3:2   | 2496x1664 |
    | 2:3   | 1664x2496 |
    | 21:9  | 3024x1296 |
    | 9:21  | 1296x3024 |

    **4K 分辨率**

    | 比例  | 像素      |
    | ----- | --------- |
    | 1:1   | 4096x4096 |
    | 4:3   | 4694x3520 |
    | 3:4   | 3520x4694 |
    | 16:9  | 5404x3040 |

    | 2:3   | 3328x4992 |
    | 21:9  | 6198x2656 |
    | 9:21  | 2656x6198 |
  </Expandable>
</ParamField>

<ParamField body="n" type="integer">
  生成图像的数量

  范围：1-15（最少1，最多15）

  默认值：1

  **注意：**

  * 实际可生成的图像数量受本参数和输入参考图像数量共同影响，参考图像数量 + 生成图像数量必须 ≤ 15
  * **必须输入纯数字（例如 `1`），不能带引号，否则会报错**

  会根据数量预先扣费
</ParamField>

<ParamField body="image_urls" type="array">
  图像到图像或图像编辑的参考图像URL列表

  * 最多10张图像
  * 图像格式：jpeg, png
  * 长宽比（宽/高）范围：\[1/3, 3]
  * 宽度和高度（像素） > 14
  * 大小不超过10MB
  * 总像素不超过6000×6000像素
</ParamField>

<ParamField body="optimize_prompt_options" type="string" default="standard">
  提示词优化模式

  * `standard`：标准模式，生成更高质量内容但处理时间较长
  * `fast`：快速模式，生成时间较短，质量一般

  默认值：`standard`
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  顺序图像生成模式（Doubao特有功能）

  控制是否生成多张图像：

  * `disabled`：关闭顺序模式，即使有多张参考图像也只生成1张图像（默认）
  * `auto`：开启顺序模式，可生成多张图像

  **使用须知：**

  * ✅ 必须提供 `image_urls` - 至少需要1张参考图像
  * ✅ 设置 `n: 3` 或使用 `sequential_image_generation: "auto"` + `max_images: 3`
  * ✅ 这将基于参考图像生成3张不同的图像
  * ⚠️ 当 `n > 1` 时会自动设为 `auto`

  **限制：**

  * 纯文本到图像（无 `image_urls`）不能生成多张图像 - 这是Doubao API限制
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  顺序图像生成选项

  当 `sequential_image_generation` 设置为 `auto` 时可用

  **属性：**

  * `max_images`（整数）：指定生成图像数量，范围为1-15

  **示例：**

  ```json
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 3
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否为生成的图像添加水印

  * `true`：添加水印
  * `false`：不添加水印（默认）
</ParamField>
