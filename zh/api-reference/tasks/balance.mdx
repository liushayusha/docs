## Authorizations

<ParamField header="Authorization" type="string" required>
  所有接口均需要使用Bearer Token进行认证

  获取 API Key：

  访问 <a href={apiKeyUrl} target="_blank">API Key 管理页面</a> 获取您的 API Key

  使用时在请求头中添加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Response

<ResponseField name="statusCode" type="number">
  HTTP 状态码，200 表示成功
</ResponseField>

<ResponseField name="message" type="string">
  提示信息，成功时为空字符串
</ResponseField>

<ResponseField name="data" type="object">
  返回数据

  <Expandable title="属性">
    <ResponseField name="credits" type="number">
      用户当前积分余额
    </ResponseField>

    <ResponseField name="free_quotas" type="array">
      各模型的免费额度信息

      <Expandable title="数组元素属性">
        <ResponseField name="model_key" type="string">
          模型标识
        </ResponseField>

        <ResponseField name="total" type="number">
          免费额度总次数
        </ResponseField>

        <ResponseField name="used" type="number">
          已使用次数
        </ResponseField>

        <ResponseField name="remaining" type="number">
          剩余可用次数
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="ok" type="boolean">
  请求是否成功
</ResponseField>

  ```swift Swift
  import Foundation

  let url = URL(string: "https://aireiter.com/api/openapi/balance")!
  var request = URLRequest(url: url)
  request.httpMethod = "GET"
  request.setValue("Bearer <token>", forHTTPHeaderField: "Authorization")

  let task = URLSession.shared.dataTask(with: request) { data, response, error in
      if let error = error {
          print("Error: \(error)")
          return
      }

      if let data = data, let responseString = String(data: data, encoding: .utf8) {
          print(responseString)
      }
  }

  task.resume()
  ```
